# 感觉皮层可塑性分析与应用

## 何时使用
- 用户描述截肢后幻肢感或感觉映射变化
- 设计针对感官缺失人群（如盲人、聋人）的康复训练方案
- 评估强化感觉训练对皮层功能的影响
- 解释皮层切除或损伤后的功能代偿现象

## 执行步骤
1. **识别感觉输入变化类型**：判断是截肢、感觉训练、皮层切除，还是其他导致感觉输入显著改变的因素。
2. **确定适用对象**：确认对象为灵长类动物（包括人类）、截肢个体或接受感官训练者，且不在急性不可逆损伤期。
3. **预测皮层重组方向**：
   - 若为截肢，邻近手指/身体部位的皮层代表区将扩张占据原区域。
   - 若为皮层切除，原对应身体部位的感觉投射将移向切除区周围皮层。
   - 若为感觉训练，受训部位的皮层代表区将扩大。
4. **考虑跨模态重组**：在盲人或聋人中，视皮层或听皮层可能被用于处理非原始模态信息（如触觉或视觉周边刺激）。
5. **评估时间窗口**：确认变化发生在数天至数周内，属于功能性重组，需给予足够观察或干预周期。
6. **生成康复建议**：基于重组机制，推荐假肢感觉反馈训练、感官替代设备使用或针对性感觉辨别练习。

> 提示：本技能输出为预警型（Alert），强调神经系统具备动态重组能力，可用于指导临床或训练干预。
