# 婴儿语言发育阶段划分

## 何时使用
- 用户描述婴儿的语言行为（如发声、咿呀学语、首次说话）
- 需要判断婴儿所处的语言发育阶段以指导养育或干预
- 用户提供婴儿月龄并询问是否符合典型语言发展轨迹

## 执行步骤
1. **确认关键转折点**：判断婴儿是否已“有意识地说出一个能被理解的单词”（如“妈妈”“灯”“手”）。该事件通常发生在10～12个月。
2. **若尚未说出有意义单词**：婴儿处于**语言准备期**，特征包括：
   - 出生后通过啼哭发声
   - 随后发出无意义元音（a、o、u、e）和辅音（n、k、p、m）
   - 约5个月进入咿呀学语阶段（如ba-ba、ma-ma），但无实际指代意义
   - 逐步保留环境中存在的语言声音，淘汰无关声音
3. **若已说出有意义单词**：婴儿进入**语言发展期**，特征包括：
   - 词汇量持续增长
   - 句子中代词使用增多
   - 5～6岁时词汇达2000～3000个，可叙述事件
   - 7岁左右出现内部语言（如默读、默想）
4. **返回结果**：明确指出当前阶段为“语言准备期”或“语言发展期”，用于支持早期教育决策或临床评估。

> 注意：本技能仅适用于处于正常人类语言环境中的婴儿，不适用于非人类灵长类或极端语言剥夺环境。
