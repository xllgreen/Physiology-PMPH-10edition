# 前庭性眼震颤评估

## 何时使用
- 患者接受头位旋转等角加速度刺激后出现眼震颤
- 需要通过旋转试验评估前庭功能状态
- 区分生理性眼震颤与病理性眼震颤（如静息状态下出现的眼震颤）

## 执行步骤

1. **确认前提条件**：确保患者前庭器官结构完整、中枢神经系统功能正常，并存在角加速度刺激（如开始或停止旋转）。

2. **实施旋转刺激**：让患者头前倾30°，绕身体纵轴在20秒内旋转10圈后突然停止。

3. **观察眼震颤特征**：
   - 以**快动相方向**作为眼震颤方向（因其更易观察）
   - 记录眼震颤**持续时间**（正常为20～40秒）
   - 记录眼震颤**频率**（正常为5～10次/秒）

4. **判断功能状态**：
   - **持续时间过长** → 前庭功能过敏
   - **持续时间过短或消失** → 前庭功能减弱或病变
   - **静息状态下出现眼震颤** → 提示病理性原因（如脑干损伤），不属本技能适用范围

5. **注意约束条件**：
   - 匀速旋转阶段不会诱发眼震颤
   - 病理性眼震颤不依赖角加速度刺激，需转诊神经科进一步评估

## 输出结果
输出眼震颤方向（以快动相为准）、持续时间、频率，并据此判断前庭功能状态。
