# 血浆渗透压组成与生理作用分析

## 何时使用
- 评估患者是否存在因血浆蛋白降低（如肝病、肾病综合征、营养不良）导致的组织水肿风险
- 分析电解质异常（如高钠血症、低钠血症）对细胞体积的影响
- 解释毛细血管处液体滤过与重吸收失衡的机制
- 判断体液分布异常是否源于渗透压变化

## 核心操作流程
1. **确认系统存在半透膜分隔**（如细胞膜或毛细血管壁），且溶质浓度存在差异。
2. **区分两类渗透压**：
   - **晶体渗透压**：由Na⁺、Cl⁻等小分子晶体物质形成，占血浆总渗透压绝大部分（约300 mOsm/(kg·H₂O)），主导细胞内外水平衡。
   - **胶体渗透压**：主要由白蛋白形成，仅约1.3 mOsm/(kg·H₂O)，但对维持血浆容量和调节血管内外水平衡至关重要。
3. **评估关键贡献比例**：
   - Na⁺和Cl⁻贡献晶体渗透压约80%。
   - 白蛋白贡献胶体渗透压75%～80%。
4. **判断病理影响**：
   - 若血浆蛋白下降 → 胶体渗透压降低 → 毛细血管滤过增加 → 组织水肿。
   - 若血浆晶体物质浓度异常 → 晶体渗透压改变 → 水在细胞内外重新分布 → 细胞肿胀或皱缩。
5. **输出结论**：明确渗透压变化类型、受影响的体液 compartment（细胞内液、组织液或血浆），并关联临床表现。

> 注意：本技能不适用于无半透膜的均质溶液系统或非水溶剂体系。
