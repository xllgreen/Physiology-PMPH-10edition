# 铁吸收与储存平衡机制分析

## 何时使用本技能
- 当需要解释食物中铁如何被人体吸收时
- 当评估缺铁性贫血的潜在机制（如胃大部切除术后）
- 当分析补铁治疗效果不佳的原因
- 当判断铁代谢是否处于稳态平衡

> **注意**：本技能不适用于已知存在遗传性血色病、铁过载或其他铁代谢紊乱疾病的情况。

## 执行步骤

1. **确认铁的可吸收形式**：确保食物中的铁已被还原为亚铁离子（Fe²⁺），此过程依赖胃酸和维生素C。若胃酸分泌不足（如胃大部切除术后），铁吸收将显著减少。

2. **启动顶端膜摄取**：当Fe²⁺到达小肠上部时，通过二价金属转运体1（DMT1）以H⁺依赖的主动转运方式进入上皮细胞。

3. **执行基底侧外排**：细胞内Fe²⁺经铁转运蛋白1（FPN1）从基底侧膜输出入血，FPN1是哺乳动物唯一已知的铁外排通道。

4. **管理细胞内储存**：未立即外排的Fe²⁺被氧化为Fe³⁺，并与脱铁铁蛋白结合形成铁蛋白（ferritin），在细胞内暂存。

5. **维持动态平衡**：
   - 铁蛋白中的铁可缓慢释放入血供机体利用；
   - 未利用的铁随肠上皮细胞更新（每2～5天）脱落排出；
   - 缺铁状态下，整体吸收率可提高2～5倍。

6. **量化每日平衡**：成人每日净吸收约1mg铁，与生理丢失量基本持平，用于维持铁稳态。

## 关键变量说明
- **Fe²⁺**：唯一可被高效吸收的铁形式
- **DMT1**：顶端膜H⁺依赖的铁摄取转运体
- **FPN1**：基底侧膜铁外排蛋白
- **铁蛋白**：细胞内安全储存铁的形式

完成上述分析后，可输出“铁吸收量（mg/天）”作为诊断或疗效评估依据。
