# 网织红细胞计数解读以评估骨髓造血功能

## 何时使用
- 需评估骨髓红系造血活性（如贫血病因鉴别）；
- 监测急性失血或溶血性贫血后的骨髓代偿反应；
- 判断促红细胞生成素（EPO）治疗是否有效；
- 怀疑再生障碍性贫血或骨髓抑制状态。

## 执行步骤
1. **确认适用条件**：
   - 患者存在有效红细胞生成过程；
   - 已获取合格的外周血样本；
   - 排除纯红细胞再生障碍或骨髓完全衰竭（此时本指标无效）。

2. **获取网织红细胞数据**：
   - 同时记录网织红细胞百分比和绝对值（推荐使用绝对值，更准确）。

3. **对照参考范围判断**：
   - 正常范围：0.5%～1.5%（对应绝对值约25–75×10⁹/L）；
   - 显著升高：>1.5%，尤其达30%～50%提示骨髓红系高度活跃；
   - 降低：<0.5%提示骨髓造血低下。

4. **结合临床情境解读**：
   - **升高**：见于失血后恢复期、溶血性贫血、EPO治疗有效反应、高原适应初期；
   - **正常或降低伴贫血**：提示骨髓造血功能低下（如再生障碍性贫血、化疗后骨髓抑制）；
   - **肾性贫血患者无升高**：支持EPO缺乏诊断。

5. **注意事项**：
   - 避免仅依赖百分比，应计算绝对值；
   - 注意药物影响（如化疗药可抑制网织红细胞生成）；
   - 网织红细胞寿命仅1～2天，反映近期（24–48小时内）造血状态。
