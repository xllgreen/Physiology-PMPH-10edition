# 躯体感觉分类与传入通路分析

## 何时使用
- 分析机械、温度或伤害性刺激引发的感觉类型
- 解释本体感觉（位置觉、运动觉）的来源
- 区分躯体感觉与内脏/特殊感觉（如视觉）

## 如何执行
1. 判断刺激来源：
   - **皮肤** → 浅感觉（触-压、温度、痛、痒）
   - **肌肉/肌腱/关节** → 深感觉（本体感觉）
2. 确认传入通路完整性：
   - 初级传入神经元胞体位于背根神经节（DRG）或脑神经节
   - 周围突连接感受器，中枢突进入脊髓/脑干
3. 关联分子机制（如适用）：
   - 温度觉：TRPV1（热）、TRPM8（冷）
   - 触-压觉：Piezo机械门控通道
4. 输出感觉类型及传导路径，用于解释感知形成或诊断感觉障碍

> 注意：此技能不适用于内脏感觉或特殊感觉系统。

感受器与通道对应关系见 `references/somatosensory-receptors.md`。
