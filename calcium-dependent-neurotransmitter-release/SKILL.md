# 钙依赖性神经递质释放技能

## 何时使用
- 动作电位传至突触前末梢并引发膜去极化
- 需要解释突触传递的分子机制
- 分析钙通道阻滞剂、Mg²⁺浓度变化或突触蛋白突变对递质释放的影响
- 研究囊泡循环（如全坍塌 vs. 触-弹模式）

## 如何执行
1. **确认前提条件**：
   - 动作电位已到达突触前末梢
   - 细胞外存在Ca²⁺
   - 囊泡已完成递质装载

2. **启动钙内流**：
   - 去极化激活电压门控钙通道 → Ca²⁺快速内流 → 轴浆[Ca²⁺]升高

3. **执行囊泡释放五步流程**：
   a) **动员**：Ca²⁺-CaM-CaMKⅡ磷酸化突触蛋白，使囊泡从细胞骨架解离
   b) **摆渡**：Rab3/Rab27介导囊泡向活性区移动
   c) **锚定**：v-SNARE（VAMP/synaptobrevin）与t-SNARE（syntaxin + SNAP-25）形成复合物
   d) **融合**：突触结合蛋白（synaptotagmin）感知Ca²⁺后诱导膜融合，形成并扩大融合孔（1nm → 50nm）
   e) **出胞**：以量子式释放递质

4. **处理后续事件**：
   - 在0.2～0.5ms内完成释放
   - 启动囊泡回收（全坍塌或触-弹模式）
   - 通过Na⁺-Ca²⁺反向转运体清除Ca²⁺

5. **输出结果**：生成突触后电位，可用于解释药理或病理干预效应
