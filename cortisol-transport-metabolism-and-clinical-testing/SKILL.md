# 皮质醇的运输、代谢与临床检测指标

## 何时使用
- 需评估肾上腺皮质功能（如疑似库欣综合征或肾上腺皮质功能减退）
- 患者存在肝病、肾病、妊娠或接受雌激素治疗等可能干扰皮质醇检测结果的情况
- 需解释总皮质醇与游离皮质醇检测结果不一致的原因

## 执行步骤

1. **确认皮质醇在血液中的结合状态**：
   - 约90%与皮质类固醇结合球蛋白（CBG）结合
   - 少量与白蛋白结合
   - 仅5–10%为具有生物活性的游离型

2. **理解动态平衡与生物活性**：
   - 结合型与游离型皮质醇可相互转化，维持稳态
   - 仅游离皮质醇能穿过细胞膜进入靶细胞发挥作用

3. **评估代谢与排泄路径**：
   - 主要在肝脏降解失活
   - 约70%代谢产物为17-羟类固醇，经尿排泄
   - 约15%以原形经胆汁排泄，少量原形从尿排出

4. **选择合适的临床检测方法**：
   - 避免单独依赖尿17-羟类固醇（易受药物、肝病等干扰，特异性低）
   - 优先选择24小时尿游离皮质醇检测（不受CBG影响，敏感性与特异性更高）

5. **考虑关键约束条件**：
   - 严重肝病或肾病时，尿代谢物测定不可靠
   - 妊娠或雌激素治疗可升高CBG，导致总皮质醇升高但游离皮质醇不变

6. **解读结果**：
   - 24小时尿游离皮质醇高于正常提示皮质醇分泌过多（如库欣综合征）
   - 低于正常提示肾上腺皮质功能减退
