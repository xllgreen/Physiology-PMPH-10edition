# 记忆巩固的神经生物学机制

## 何时使用
- 用户询问为何重复复习有助于长期记忆
- 需要解释脑震荡或麻醉导致特定事件无法被记住的机制
- 分析海马损伤患者为何保留旧记忆但无法形成新长期记忆
- 探讨蛋白质合成抑制剂对学习效果的影响

## 执行步骤
1. **确认前提条件**：确保信息已进入短时程记忆系统，且存在神经元反复激活（如通过复习）。
2. **识别触发条件**：若新信息被反复激活，则启动记忆巩固过程。
3. **追踪突触变化**：观察以下突触层面的适应性改变：
   - 新突触形成
   - 已有突触体积增大
   - 突触后膜受体数量增加
   - 受体敏感性增强
4. **监测钙信号通路**：确认突触后神经元内钙离子（Ca²⁺）浓度升高，激活第二信使系统。
5. **评估蛋白质合成窗口**：在学习后0–4小时内，新蛋白质合成对晚期长时程增强（L-LTP）至关重要；超过此窗口则干预无效。
6. **验证海马作用**：确认海马结构完整，因其负责将短时程记忆整合并转移至皮层进行长期存储。
7. **检查阻断因素**：若在记忆巩固关键期发生脑震荡或深度麻醉，立即终止该过程，导致记忆丢失。

## 输出结果
- 成功：形成长时程记忆，可稳定存储并长期提取
- 失败：信息被遗忘，无法进入长期存储
