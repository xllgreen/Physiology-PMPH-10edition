# 微循环血流量的局部代谢调节

## 适用对象
- 后微动脉
- 毛细血管前括约肌
- 真毛细血管
- 代谢活跃组织（如运动中的骨骼肌）

> **注意**：本机制不适用于受神经直接支配的大动脉；毛细血管本身无平滑肌，不受神经直接控制。

## 触发条件
当局部组织出现以下任一情况时触发血管舒张：
- O₂分压降低（低氧状态）
- 代谢产物（如CO₂、H⁺、腺苷、K⁺、乳酸等）积聚

## 执行流程

1. **识别基础节律活动**
   - 确认后微动脉和毛细血管前括约肌存在每分钟5～10次的交替性收缩与舒张（称为“血管运动”）
   - 此节律控制真毛细血管的周期性开放与关闭

2. **判断当前状态**
   - 若括约肌处于**收缩期**：真毛细血管关闭 → 组织代谢产物积聚 + O₂分压下降
   - 若括约肌处于**舒张期**：血流恢复 → 清除代谢废物 + 补充O₂与营养

3. **执行调节动作**
   - 检测到低氧或代谢产物积聚 → 直接作用于后微动脉和毛细血管前括约肌 → 引起舒张
   - 血流恢复后，代谢产物被清除 → 括约肌再次收缩 → 关闭毛细血管

4. **评估功能效果**
   - 安静状态下仅20%–35%毛细血管开放
   - 代谢增强时（如运动），开放比例显著上升 → 增大交换面积、缩短扩散距离、提升微循环血流量

5. **确认调节性质**
   - 此过程为**自身调节**，不依赖神经或体液系统
   - 调节目标：按需分配血流，保障组织供氧与废物清除

## 关键变量监控
- **血管运动频率**：正常5–10次/分钟
- **开放毛细血管比例**：活动状态 >35%
- **关键代谢信号**：O₂↓、CO₂↑、H⁺↑、腺苷↑、K⁺↑

> 使用本技能时，应聚焦于局部代谢与血管反应的因果关系，避免引入神经或激素调节机制。
