# 消化间期移行性复合运动（MMC）周期分析

## 何时使用
- 用户询问空腹期间胃肠如何维持清洁与运动节律
- 需要解释MMC四个时相的特征与持续时间
- 评估MMC减弱对功能性消化不良或SIBO的潜在影响
- 分析胃动素或一氧化氮在MMC调控中的作用

## 执行步骤
1. **确认适用条件**：确保对象处于消化间期（空腹状态超过数小时，无近期进食）。
2. **识别MMC四相结构**：
   - **Ⅰ相（静息期）**：持续45～60分钟，仅慢波电位，无收缩；可能由一氧化氮（NO）维持。
   - **Ⅱ相**：持续30～45分钟，出现不规则锋电位与蠕动。
   - **Ⅲ相**：持续5～10分钟，成簇锋电位引发强力规则收缩；由胃动素触发。
   - **Ⅳ相**：约5分钟，为Ⅲ相到下一Ⅰ相的过渡期。
3. **追踪传播路径**：MMC起始于胃体上部，沿小肠向远端传播。
4. **阐明生理功能**：
   - 维持空腹期断续运动
   - Ⅲ相发挥“清道夫”作用，清除残留物、脱落细胞、细菌等
   - 抑制肠道细菌过度繁殖
5. **关联病理状态**：若MMC减弱，提示可能存在功能性消化不良或小肠细菌过度生长（SIBO）。

> 注意：本技能不适用于餐后状态；进食会立即终止MMC周期。
