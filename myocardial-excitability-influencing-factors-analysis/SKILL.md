# 心肌兴奋性影响因素分析

## 何时使用
- 需判断某因素（如高钾血症、低钙血症、抗心律失常药等）对心肌兴奋性的影响
- 面临心电生理相关临床或教学问题，需解释兴奋性变化的机制
- 分析快反应细胞（如心室肌）或慢反应细胞（如窦房结）在特定条件下的兴奋性改变

## 执行步骤

1. **确认适用对象**：仅适用于具有产生动作电位能力的心肌细胞（包括快反应与慢反应细胞），排除已死亡或严重损伤细胞。

2. **确定兴奋性衡量标准**：兴奋性与刺激阈值成反比——阈值越低，兴奋性越高。

3. **按三类机制逐一分析**：
   - **静息电位或最大复极电位水平变化**（阈电位不变时）：
     - 静息电位超极化 → 兴奋性降低
     - 静息电位轻度去极化（如细胞外K⁺轻度升高）→ 兴奋性升高
     - 静息电位显著去极化（如血清K⁺ > 7–9 mmol/L）→ 钠通道失活 → 兴奋性降低或丧失
   - **阈电位水平变化**（静息电位不变时）：
     - 阈电位上移（如奎尼丁作用）→ 兴奋性降低
     - 阈电位下移（如低血钙）→ 兴奋性升高
     - 注：生理状态下阈电位通常稳定
   - **0期去极化离子通道状态**：
     - 快反应细胞依赖钠通道，慢反应细胞依赖L型钙通道
     - 仅当通道处于“静息（备用）”状态时可被激活
     - 药物或电压变化可影响通道激活/失活/复活过程，从而改变兴奋性

4. **综合判断主导机制**：结合干预类型（电解质、药物、神经递质等）和细胞类型，指出主要作用路径。

5. **输出结论**：明确说明该因素是升高还是降低心肌兴奋性，并指明主导机制类别（静息电位 / 阈电位 / 通道状态）。
