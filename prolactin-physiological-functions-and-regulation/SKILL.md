# 催乳素（PRL）的多效性生理功能与分泌调控

## 何时使用
- 患者出现泌乳、闭经、不孕或性功能减退，怀疑与PRL异常相关
- 需解释妊娠期不泌乳而产后泌乳的激素机制
- 评估应激或免疫状态对PRL的影响
- 分析药物（如多巴胺激动剂/拮抗剂）对PRL分泌的作用

## 执行步骤

### 1. 判断PRL作用的前提条件
- 确认靶组织存在PRL受体（乳腺、卵巢、睾丸、免疫细胞等）
- 评估雌/孕激素水平（乳腺作用）或睾酮水平（男性生殖作用）

### 2. 分析PRL在不同系统的生理功能
- **乳腺调节**：
  - 妊娠期：PRL协同雌激素、孕激素等促进腺泡发育，但高雌孕激素抑制泌乳
  - 产后：雌孕激素骤降 + PRL受体数量激增 → 启动并维持泌乳
- **性腺调节（双相作用）**：
  - 女性：低浓度PRL促进雌/孕激素分泌；高浓度抑制GnRH → FSH/LH下降 → 无排卵、闭经、不孕
  - 男性：在睾酮存在下，PRL促进前列腺/精囊生长；慢性高PRL血症导致睾酮下降、少精、性欲减退
- **应激与免疫**：
  - 应激时PRL与ACTH、GH同步升高
  - PRL增强淋巴细胞增殖及IgM/IgG分泌；部分免疫细胞可自产PRL

### 3. 评估PRL分泌调控机制
- 主要受下丘脑多巴胺（PIH）持续抑制
- PRF（催乳素释放因子）可能包括PrRP、TRH、VIP、5-HT等
- 吸吮刺激 → 抑制多巴胺释放 + 促进PRF → PRL大量分泌
- 高PRL通过负反馈增强多巴胺能神经元活性，抑制自身及GnRH分泌

### 4. 识别关键约束条件
- 妊娠期高雌孕激素抑制PRL的泌乳作用
- 慢性高PRL血症抑制性腺功能，导致生殖障碍
