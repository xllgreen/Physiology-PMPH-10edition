# 胎盘激素功能评估与临床应用

## 适用条件
- 妊娠已进入第6周或之后
- 胚泡已完成着床
- 胎盘结构初步形成
- 不适用于妊娠10周前以黄体为主导的阶段

## 执行步骤

### 1. 分析人绒毛膜促性腺激素（hCG）
- **确认来源**：合体滋养层细胞。
- **评估时间动态**：着床后1天可测出，妊娠8～10周达高峰，产后2周内消失。
- **判断功能**：
  - 促进胚泡植入（通过免疫豁免机制）；
  - 维持妊娠黄体，使其持续分泌孕激素和雌激素。
- **临床应用**：血或尿hCG检测是诊断早期妊娠的金标准。

### 2. 分析人胎盘催乳素（hPL）
- **确认来源**：胎盘合体滋养层。
- **评估时间动态**：妊娠5周可测出，持续上升至39～40周达峰，产后7小时内消失。
- **判断功能**：主要促进胎儿生长（又称人绒毛膜促生长激素），对人类无显著催乳作用。

### 3. 分析雌激素（以雌三醇为主）
- **确认合成路径**：依赖胎儿-胎盘单位——母体和胎儿肾上腺分泌脱氢异雄酮硫酸盐 → 胎儿肝羟化 → 胎盘转化为雌三醇。
- **评估时间动态**：妊娠29周后迅速增加，末期达非孕期1000倍。
- **判断功能**：调控胎盘、子宫、乳腺及胎儿器官生长；妊娠晚期激活子宫为分娩做准备。
- **临床应用**：母体尿雌三醇水平反映胎盘功能，降低提示胎盘功能减退。

### 4. 分析孕激素
- **确认来源转换**：妊娠第6周起由胎盘分泌，10周后完全替代卵巢黄体。
- **判断功能**：维持子宫静息状态，抑制宫缩。
- **理解分娩机制**：灵长类动物分娩启动不依赖孕激素水平下降，而是出现“功能性孕激素撤退”（受体信号通路异常），导致子宫激活。

## 输出结果
根据孕周和激素水平，综合评估：
- 妊娠是否正常维持
- 胎盘功能是否充足
- 是否存在早产或胎盘功能减退风险
- 分娩启动的生理机制是否被激活
