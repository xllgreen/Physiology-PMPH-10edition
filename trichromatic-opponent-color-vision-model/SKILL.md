# 三原色与对比色混合色觉模型

## 适用场景
- 分析正常人眼对颜色的感知机制
- 解释红绿色盲等遗传性色觉障碍的成因
- 理解颜色对比现象（如蓝块在黄背景上更蓝）
- 判断色觉模型是否适用于特定光照或病理条件

## 执行步骤

1. **确认适用条件**：
   - 用户对象必须具备正常视锥细胞功能（排除全色盲）
   - 光照条件需为明视觉（非弱光环境）
   - 涉及颜色分辨或对比任务

2. **应用三原色学说（感光层机制）**：
   - 识别三类视锥细胞的光谱响应：
     - L型（红）：峰值564 nm
     - M型（绿）：峰值534 nm
     - S型（蓝）：峰值420 nm
   - 计算任一波长光（380–760 nm）对三类细胞的激活比例
   - 输出初步颜色信号（约150种可分辨色觉）

3. **应用对比色学说（神经通路机制）**：
   - 将三原色信号转换为两对拮抗通道：红/绿、蓝/黄
   - 分析颜色对比增强效应（如边缘对比、背景影响）
   - 结合水平细胞及视觉皮层的拮抗反应解释主观色觉

4. **整合两级模型输出**：
   - 最终色觉 = 光谱成分分析（三原色） + 颜色对立处理（对比色）
   - 若存在色觉障碍，根据缺失/减弱的视锥类型调整模型

5. **验证遗传与生理约束**：
   - 红/绿视蛋白基因位于X染色体 → 男性更易患红绿色盲
   - 蓝视蛋白基因位于第7号染色体 → 蓝色觉障碍较少见且无性别差异

> 注意：本模型不适用于全色盲患者或暗视觉条件（此时仅视杆细胞活跃）。
