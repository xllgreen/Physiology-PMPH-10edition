# 分析抗利尿激素对水重吸收的调节机制

## 何时使用
- 需解释血浆渗透压升高或血容量减少如何导致尿液浓缩
- 分析ADH相关疾病（如尿崩症、SIADH）的病理生理
- 理解水通道蛋白AQP2在水调节中的核心作用

## 执行步骤
1. **确认触发条件**：
   - 血浆晶体渗透压升高 1%～2%，或
   - 循环血量减少 ≥5%～10%

2. **分析急性调节机制（几分钟内）**：
   - ADH 与集合管主细胞基底侧膜 V₂ 受体结合
   - 激活 cAMP-PKA 通路
   - AQP2 囊泡转位至顶端膜 → 水通透性↑ → 水重吸收↑

3. **分析长期调节机制（数小时至数天）**：
   - 持续高 ADH → AQP2 基因转录↑ → AQP2 蛋白合成↑ → 最大重吸收能力↑

4. **预测尿液变化**：
   - ADH ↑ → 尿量↓、尿渗透压↑（浓缩尿）
   - ADH ↓ → 尿量↑、尿渗透压↓（稀释尿）

> ⚠️ 注意：本机制要求 V₂ 受体功能正常且无乙醇等抑制因素。
