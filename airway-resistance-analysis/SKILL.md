# 气道阻力分析

## 何时使用
- 评估患者呼吸费力程度
- 解释支气管哮喘症状（尤其是呼气困难）
- 分析阻塞性通气功能障碍的生理基础
- 判断药物（如β₂激动剂）的作用机制

## 执行步骤
1. **确认适用条件**：确保存在气体流动、可测量压力差与流量，且气道结构完整；不适用于静态无气流状态。
2. **识别主要阻力来源**：按解剖部位评估贡献比例——鼻（约50%）、声门（约25%）、气管与支气管（约15%）、细支气管（<2mm，约10%）。
3. **分析关键影响因素**：
   - **气道口径**：检查跨壁压变化（吸气时阻力下降，呼气时上升）及肺实质牵引作用。
   - **自主神经调节**：判断当前神经紧张性（迷走主导→阻力↑；交感主导→阻力↓）。
   - **化学介质**：识别是否存在收缩剂（如组胺、白三烯）或舒张剂（如儿茶酚胺、PGE₂）。
4. **关联呼吸周期变化**：
   - 吸气时：跨壁压↑ + 肺牵引↑ + 交感活动↑ → 阻力↓
   - 呼气时：跨壁压↓ + 肺牵引↓ + 迷走活动↑ → 阻力↑
5. **提出干预方向**：根据阻力来源选择措施——使用β₂激动剂扩张气道、减少分泌物、解除外部压迫或控制炎症。
