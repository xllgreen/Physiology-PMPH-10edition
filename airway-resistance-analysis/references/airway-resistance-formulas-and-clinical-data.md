# 气道阻力详细参数与临床数据

## 计算公式
气道阻力（Raw） = (大气压 - 肺内压) / 气体流量
单位：cmH₂O·s/L
健康成人平静呼吸时正常值：1～3 cmH₂O·s/L

## 解剖分布详表
| 部位 | 阻力占比 | 特点 |
|------|--------|------|
| 鼻 | ~50% | 可通过鼻塞显著增加总阻力 |
| 声门 | ~25% | 受喉部肌肉调控，情绪或喉痉挛可影响 |
| 气管与主支气管 | ~15% | 管径大，阻力相对小但病变影响显著 |
| 细支气管（<2mm） | ~10% | 数量多、总横截面积大，但哮喘时可因平滑肌收缩导致阻力剧增 |

## 自主神经受体机制
- **副交感（迷走）**：ACh → M₃受体 → Gq通路 → 平滑肌收缩 → 阻力↑
- **交感**：去甲肾上腺素/肾上腺素 → β₂受体 → Gs通路 → cAMP↑ → 平滑肌舒张 → 阻力↓

## 化学介质分类
- **收缩剂**：组胺、白三烯C₄/D₄/E₄、PGF₂α、内皮素-1、乙酰胆碱
- **舒张剂**：肾上腺素、异丙肾上腺素、沙丁胺醇（选择性β₂激动剂）、PGE₂、NO

## 哮喘患者特殊表现
- 呼气时胸内压升高 → 跨壁压减小 → 小气道塌陷倾向增强
- 炎症导致黏膜水肿 + 黏液栓 → 有效管径进一步缩小
- 结果：呼气阻力显著高于吸气，表现为呼气延长、哮鸣音
