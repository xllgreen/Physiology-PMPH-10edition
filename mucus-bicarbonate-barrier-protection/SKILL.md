# 黏液-碳酸氢盐屏障的保护机制

## 何时使用
- 分析胃黏膜如何抵御高浓度H⁺和胃蛋白酶的损伤
- 解释消化性溃疡的病理生理基础
- 评估酒精、阿司匹林、NSAIDs或幽门螺杆菌感染对胃屏障的影响

## 执行步骤
1. **确认屏障组成**：识别参与屏障形成的细胞类型——胃黏膜上皮细胞、泌酸腺、贲门腺和幽门腺。
2. **评估黏液层状态**：检查黏液是否正常分泌，厚度是否维持在约500μm，并具有高黏滞性（为水的30–260倍）。
3. **验证HCO₃⁻来源**：确认HCO₃⁻由非泌酸细胞分泌，并滞留于黏液层靠近黏膜的一侧。
4. **分析pH梯度形成**：
   - 胃腔侧pH ≈ 2.0（强酸性）
   - 黏膜侧pH ≈ 7.0（中性）
   - H⁺向黏膜扩散时被黏液高黏度显著减缓
   - 扩散中的H⁺与HCO₃⁻发生中和反应：H⁺ + HCO₃⁻ → H₂CO₃ → CO₂ + H₂O
5. **检查紧密连接完整性**：确认胃黏膜上皮细胞间的紧密连接未受损，防止H⁺经细胞旁路侵入。
6. **判断屏障功能状态**：若上述任一环节受损（如受NSAIDs、酒精或幽门螺杆菌影响），则屏障失效，易导致溃疡。

> **注意**：该机制仅在黏液分泌正常、HCO₃⁻供应充足且无外源性破坏因素时有效。
