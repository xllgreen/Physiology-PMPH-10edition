# 心室泵血时相分析

## 适用条件
- 对象：左心室、右心室或哺乳动物完整心脏
- 前提：心脏瓣膜结构完整，心肌具备正常收缩与舒张功能
- 禁忌：严重瓣膜病变、心肌坏死或心室颤动等心律极度紊乱状态

## 执行步骤

### 1. 判断当前处于收缩期还是舒张期
- 若心室开始收缩 → 进入**心室收缩期**（约0.25秒）
- 若心室开始舒张 → 进入**心室舒张期**（约0.45秒）

### 2. 收缩期细分判断
- **等容收缩期**：房室瓣已关闭、半月瓣未开放 → 心室封闭、容积不变、压力快速上升
- **射血期**：
  - *快速射血期*：半月瓣开放，血液快速射入动脉，射出量约占总搏出量2/3
  - *减慢射血期*：射血速度减慢，但因血流动能仍可继续射血，此时室内压可能略低于动脉压

### 3. 舒张期细分判断
- **等容舒张期**：半月瓣已关闭、房室瓣未开放 → 心室再次封闭、容积不变、压力快速下降
- **充盈期**：
  - *快速充盈期*：房室瓣开放，心室负压抽吸血液，充盈量约占总充盈量2/3
  - *减慢充盈期*：压力梯度减小，血流速度减慢
  - *心房收缩期*：心房收缩推动额外血液入心室，贡献约25%充盈量，达舒张期末容积（EDV）

### 4. 输出结果
明确当前时相，并说明：
- 瓣膜开闭状态
- 心室、心房、动脉三者之间的压力关系
- 容积变化趋势
- 血流方向及动力来源（如心肌收缩、血流惯性、心室抽吸或心房收缩）
