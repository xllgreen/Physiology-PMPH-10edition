# 基于清除率推测肾小管转运功能

## 何时使用
- 已知某物质的清除率（Cx）和肾小球滤过率（GFR）
- 该物质可通过肾小球滤过（不适用于完全不被滤过的物质）
- 需要判断肾小管对该物质是净重吸收、净分泌，还是两者兼有但以某一过程为主

## 执行步骤
1. 获取该物质的清除率（Cx，单位：ml/min）和肾小球滤过率（GFR，单位：ml/min）。
2. 比较 Cx 与 GFR：
   - 若 **Cx ≈ 0**，表明该物质被肾小管全部重吸收，存在「净重吸收」。
   - 若 **Cx < GFR**，表明该物质在滤过后被肾小管和集合管「净重吸收」（重吸收量 > 分泌量）。
   - 若 **Cx > GFR**，表明肾小管必定存在「净分泌」（分泌量 > 重吸收量）。
3. 注意：清除率仅反映「净转运」结果，无法单独量化重吸收或分泌的绝对量，只能判断净方向。
4. 输出结论：明确说明该物质在肾小管的净转运类型。

> 提示：此方法适用于葡萄糖、尿素、肌酐、对氨基马尿酸等可滤过溶质。
