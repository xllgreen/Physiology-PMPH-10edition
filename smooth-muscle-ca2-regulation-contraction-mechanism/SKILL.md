# 平滑肌Ca²⁺调控与收缩机制

## 何时使用
- 分析平滑肌对激素、神经递质或电刺激的收缩响应
- 解释为何平滑肌收缩缓慢且持续时间长
- 对比平滑肌与骨骼肌/心肌的收缩机制差异
- 理解药物（如钙通道阻滞剂、IP₃通路激动剂）对平滑肌张力的影响

## 执行步骤

### 1. 判断Ca²⁺来源途径
- **电-机械耦联**：动作电位激活电压门控或机械门控Ca²⁺通道 → Ca²⁺主要来自胞外，少量经RyR从肌浆网（SR）释放。
- **药物-机械耦联**：化学信号（如激素）激活G蛋白偶联受体 → 激活PLC → 生成IP₃ → IP₃结合IP₃R → SR释放Ca²⁺（无需动作电位）。

### 2. 触发收缩
- 胞质Ca²⁺与钙调蛋白（CaM）结合形成Ca²⁺-CaM复合物。
- Ca²⁺-CaM激活肌球蛋白轻链激酶（MLCK）。
- MLCK磷酸化肌球蛋白轻链（MLC，20kD）→ 提高横桥ATP酶活性 → 启动肌丝滑行 → 收缩。

### 3. 实现舒张
- 胞质Ca²⁺浓度下降（通过SR钙泵、Na⁺-Ca²⁺交换体、膜钙泵清除Ca²⁺）。
- MLCK失活。
- 肌球蛋白轻链磷酸酶（MLCP）使MLC去磷酸化 → 横桥脱离 → 肌肉舒张。

### 4. 注意关键特征
- 不依赖肌钙蛋白（TnC），无横纹结构。
- 舒张过程缓慢（因Ca²⁺清除效率低）。
- 粗肌丝横桥多向伸出，滑行范围大，舒缩幅度广。
- 以致密体和致密斑替代Z盘传递张力。
