# 心输出量与心功能评估

## 何时使用
- 需评估个体在静息或运动状态下的心脏泵血功能
- 需比较不同体型者（如儿童 vs 成人）的心功能 → 使用心指数（CI）
- 需判断心力衰竭患者的代偿能力或运动耐量
- 已知心率（HR）、舒张期末容积（EDV）、收缩期末容积（ESV）或搏出量（SV）

## 执行步骤
1. **计算基础参数**：
   - 搏出量（SV） = EDV - ESV
   - 射血分数（EF） = (SV / EDV) × 100%
   - 心输出量（CO） = HR × SV

2. **标准化心输出量**（若需跨体型比较）：
   - 心指数（CI） = CO / 体表面积（BSA）

3. **评估心泵功能储备**：
   - 搏出量储备 = 舒张期储备 + 收缩期储备
   - 心率储备 = 最大有效心率 - 静息心率
   - 总储备 = 最大CO - 静息CO

4. **结合临床背景解读结果**：
   - EF < 55% 提示收缩功能受损
   - CI < 2.5 L/(min·m²) 可能提示心功能不全
   - 储备显著降低见于心力衰竭患者

> ⚠️ **限制条件**：不适用于心律失常导致无效搏动（如房颤伴快速心室率）或休克/麻醉等血流动力学不稳定状态。
