# 公式与参考值

## 核心公式
- **SV** = EDV - ESV
- **EF** = (SV / EDV) × 100%
- **CO** = HR × SV
- **CI** = CO / BSA

## 正常参考值
- **EDV**: ≈125 ml
- **ESV**: ≈55 ml
- **SV**: 60–80 ml（平均70 ml）
- **EF**: 55%–65%
- **静息CO**: 男 4.5–6.0 L/min，女低约10%
- **静息CI**: 3.0–3.5 L/(min·m²)
- **BSA**: 中等身材成人约1.6–1.7 m²

## 功能储备细节
- **舒张期储备**：EDV 可增至 ≈140 ml（+15 ml）
- **收缩期储备**：ESV 可降至 <20 ml（+35–40 ml）
- **心率储备**：
  - 静息 HR：60–100 次/分
  - 最大有效 HR：160–180 次/分（普通人群）；运动员可达 200–220 次/分
  - HR > 180 次/分 → 舒张期过短 → 充盈不足 → SV ↓ → CO ↓

## 病理状态表现
- **心力衰竭患者**：
  - 静息 CO 可能正常
  - 储备显著下降
  - 代偿性心率升高（120–140 次/分），进一步增快反而使 CO 下降
- **心房颤动**：
  - 丧失心房“初级泵”作用 → 心室充盈减少
  - 运动时 CO 上升受限
