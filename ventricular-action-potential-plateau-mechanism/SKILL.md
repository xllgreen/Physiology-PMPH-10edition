# 心室肌细胞动作电位平台期（2期）形成机制

## 适用对象
- 心室肌细胞
- 浦肯野细胞（部分相似）

## 触发条件
当心室肌细胞完成1期快速复极且膜电位接近0 mV时，启动平台期离子流平衡机制。

## 执行步骤

1. **确认平台期特征**：平台期持续100～150毫秒，是心室肌动作电位时程（APD=200～300 ms）显著延长的关键原因。

2. **识别内向电流成分**：
   - **L型钙电流（I_Ca-L）**：通过慢激活/慢失活的L型钙通道介导Ca²⁺缓慢持久内流，为主要去极化电流；
   - **晚钠电流（I_Na,Late）**：快钠通道未完全失活所致，产生持续微弱Na⁺内流；
   - **Na⁺-Ca²⁺交换电流（I_Na-Ca）**：通常为内向电流（3Na⁺入，1Ca²⁺出），在平台期贡献去极化。

3. **识别外向电流成分**：
   - **I_K1通道**：因内向整流特性，在去极化状态下通透性低，K⁺外流受限，有助于延长平台期；
   - **延迟整流钾电流（I_K）**：随时间逐渐增强，在2期早期抗衡I_Ca-L，在2期晚期主导复极启动。

4. **分析动态平衡过程**：
   - **2期早期**：Ca²⁺内流 ≈ K⁺外流 → 膜电位稳定于0 mV附近，形成平台；
   - **2期晚期**：钙通道逐渐失活 + I_K逐渐增强 → K⁺外流 > Ca²⁺内流 → 启动缓慢复极。

5. **关联临床意义**：
   - **Ⅳ类抗心律失常药**（如维拉帕米）阻断I_Ca-L → 缩短平台期 → 缩短APD并减弱收缩力；
   - **1型/2型长QT综合征**：I_K通道基因突变 → I_K减小 → APD延长；
   - **3型长QT综合征**：SCN5A突变 → I_Na,Late增加 → APD延长。

## 注意事项
- 不适用于神经元或骨骼肌细胞（无平台期）；
- 不适用于窦房结P细胞（慢反应细胞，无明显平台期）；
- 需确保L型钙通道、I_K、I_K1、I_Na-Late、I_Na-Ca等功能正常。
