# 肺泡表面张力与Laplace定律分析

## 触发条件
当需要解释或预测以下情况时启用本技能：
- 肺泡稳定性异常（如小肺泡萎缩、大肺泡过度膨胀）
- 肺顺应性显著下降
- 新生儿呼吸窘迫综合征（NRDS）的病理机制或风险评估
- 肺表面活性物质功能缺失或不足的生理后果

## 执行步骤
1. **确认适用条件**：确保存在肺泡液-气界面，且肺泡半径（r）和表面张力系数（T）可获取或可比较。
2. **应用Laplace定律**：计算肺泡回缩压强 $ P = \frac{2T}{r} $，分析不同半径肺泡间的压力差异。
3. **评估失稳风险**：若表面张力未被调节，小肺泡内压将高于大肺泡，导致气体重分布和肺泡塌陷。
4. **引入表面活性物质动态调节机制**：
   - 吸气时：肺泡扩张 → 表面活性物质密度降低 → 表面张力适度上升 → 防止过度膨胀
   - 呼气时：肺泡缩小 → 表面活性物质密度升高 → 表面张力显著下降 → 防止塌陷
5. **判断临床意义**：
   - 若为胎儿或早产儿，评估表面活性物质合成状态
   - 若为成人肺疾病状态，判断是否存在表面活性物质功能障碍
6. **输出预警**：若表面活性物质不足或Laplace效应未被补偿，应预警肺泡不稳定、肺不张或NRDS风险。

> 注意：本技能不适用于无液-气界面的情况（如肺内注入生理盐水后），也不适用于除表面活性物质完全缺失模型外的其他稳态分析。
