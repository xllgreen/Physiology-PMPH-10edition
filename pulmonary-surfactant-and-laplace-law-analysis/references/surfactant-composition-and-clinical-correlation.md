# 肺表面活性物质组成与临床关联

## 化学组成
- **脂质（90%）**：其中二棕榈酰磷脂酰胆碱（DPPC）占60%以上，是降低表面张力的关键成分。
- **表面活性蛋白（10%）**：包括SP-A、SP-B、SP-C、SP-D，参与分子铺展、免疫调节及结构稳定。

## 分子作用机制
DPPC为双嗜性分子：
- 极性端插入肺泡液体层
- 非极性端朝向肺泡腔
- 形成单分子层，随肺泡容积变化动态调整密度

## 发育时间线
- 胎儿在妊娠第6～7个月开始合成表面活性物质
- 第35周后显著增加，接近足月时达成熟水平

## 临床检测与干预
- **羊水检测**：通过L/S比值（卵磷脂/鞘磷脂）或PG（磷脂酰甘油）含量评估肺成熟度
- **产前干预**：给予糖皮质激素促进Ⅱ型肺泡细胞合成表面活性物质
- **产后治疗**：外源性表面活性物质替代疗法（如Survanta、Curosurf）

## Laplace定律推导示例
假设两个相邻肺泡，半径分别为 r₁=50 μm 和 r₂=100 μm，若表面张力 T=0.07 N/m（纯水）：
- P₁ = 2×0.07 / (50×10⁻⁶) = 2800 Pa
- P₂ = 2×0.07 / (100×10⁻⁶) = 1400 Pa
→ 气体从P₁流向P₂，小肺泡塌陷

加入表面活性物质后，T在小肺泡中可降至0.005 N/m，则：
- P₁' = 2×0.005 / (50×10⁻⁶) = 200 Pa
- P₂' ≈ 2×0.02 / (100×10⁻⁶) = 400 Pa（因密度较低，T较高）
→ 压力趋于平衡，维持稳定
