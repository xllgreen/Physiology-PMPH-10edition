# Access New-Form Digital Textbook Resources

## When to Use
- 你拥有人民卫生出版社出版的全国高等学校五年制本科临床医学专业第十轮规划教材
- 教材封底带有红标二维码和可揭开的绿标激活码
- 你需要访问与该教材配套的电子书、授课PPT、测试题、视频、AR互动等数字资源

## Prerequisites
- 带封底红标和绿标激活码的纸质教材
- 可扫码设备（如手机或平板）
- 稳定的网络连接

## Step-by-Step Procedure
1. **扫描红标二维码**：使用设备扫描教材封底红色标签上的二维码，获取《图书使用说明》文档。
2. **揭开红标**：小心揭开封底红标覆盖层，露出下方绿色标签上的激活码。
3. **扫描绿标激活码**：扫描绿标上的激活码，进入人卫账号注册/登录页面。
   - 若已有账号，请直接登录
   - 若无账号，请先完成注册
4. **自动绑定权限**：成功登录后，系统将自动为你绑定该图书的数字资源访问权限。
5. **访问数字资源**：通过以下任一方式使用服务：
   - 访问网站：https://zengzhi.ipmph.com
   - 下载并安装“人卫智网”官方应用（通过书中提供的二维码下载）
6. **使用功能**：在网页端或App中可使用目录导航、全文检索、思维导图、授课PPT、目标测试题、解题思路、动画、视频微课、AR互动、数字切片、DICOM图像等功能。
7. **获取技术支持**：如遇问题，请联系客服：
   - 电话：400-111-8166
   - 邮箱：zengzhi@pmph.com

> 注意：本流程仅适用于人民卫生出版社第十轮新形态教材，不适用于无二维码或非第十轮版本。
