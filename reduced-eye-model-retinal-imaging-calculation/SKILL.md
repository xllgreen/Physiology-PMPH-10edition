# 简化眼模型视网膜成像计算

## 何时使用
- 需要估算正视眼在良好光照条件下对某物体的视网膜成像大小；
- 判断该成像是否达到人眼分辨极限（≥4.5μm）；
- 分析物体大小与物距对视觉清晰度的联合影响。

> **注意**：本技能仅适用于屈光正常、无严重散光且已矫正的正视眼，在无调节或调节后成像清晰的状态下使用。

## 执行步骤
1. **确认输入参数**：
   - 物体实际大小（AB）；
   - 物体至眼节点的距离（Bn，即物距）；
   - 节点至视网膜距离（nb），固定为15 mm。

2. **应用相似三角形公式计算视网膜像大小（ab）**：
   ```
   ab = (AB × nb) / Bn
   ```

3. **判断视觉清晰度**：
   - 若 ab ≥ 4.5 μm，则可形成清晰视觉；
   - 若 ab < 4.5 μm，则无法被分辨。

4. **输出结果**：返回计算出的 ab 值及是否可清晰分辨的结论。

## 注意事项
- 物体大小本身不能决定清晰度，必须结合物距；
- 不适用于未矫正屈光不正或严重散光眼；
- 模型假设为单球面折光系统，前后径20 mm，折射率1.333，节点位于折射界面后5 mm处。
