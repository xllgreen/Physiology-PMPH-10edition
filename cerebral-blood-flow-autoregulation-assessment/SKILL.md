# 脑血流量自身调节范围评估

## 何时使用
- 当需要判断患者脑灌注是否稳定时
- 当平均动脉压（MAP）低于60 mmHg或高于140 mmHg（正常人）时
- 当处理高血压患者的脑循环管理时
- 当怀疑脑缺血或过度灌注相关症状时

## 执行步骤
1. **确认前提条件**：
   - 颅内压（ICP）应在正常范围（5–15 mmHg）
   - 脑血管结构完整
   - 排除严重颅内病变（如脑水肿、肿瘤）

2. **计算脑灌注压（CPP）**：
   - CPP = MAP - ICP

3. **判断MAP是否在自身调节范围内**：
   - **正常人**：若 MAP ∈ [60, 140] mmHg，则脑血流量（CBF）维持恒定（约50–60 ml/100g/min）
   - **高血压患者**：自身调节上限可能右移至180–200 mmHg，但下限也可能升高

4. **评估风险**：
   - 若 MAP < 60 mmHg（或低于个体调节下限）：预警脑缺血风险（CBF < 40 ml/100g/min 可致意识丧失）
   - 若 MAP > 140 mmHg（或高于个体调节上限）：预警脑过度灌注及脑水肿风险

5. **输出预警**：
   - 明确说明当前MAP是否超出自身调节范围
   - 指出潜在临床后果（缺血或水肿）

> 注意：此调节机制独立于神经和体液因素，是维持脑功能稳定的首要机制。
