# 氧解离曲线偏移分析

## 何时使用
当以下任一情况发生时，应用本技能：
- 血液pH或PCO₂发生变化（如代谢性/呼吸性酸中毒）
- 体温异常升高或降低（如高热、低温治疗）
- 怀疑慢性缺氧、贫血或高原暴露导致红细胞代偿
- 输注储存超过3周的库存血
- 一氧化碳（CO）中毒或疑似CO暴露

## 执行步骤

1. **确认P₅₀基准值**：正常P₅₀ ≈ 26.5 mmHg；P₅₀增大表示血红蛋白（Hb）对O₂亲和力下降，曲线右移；反之则左移。

2. **评估各影响因素**：
   - **波尔效应（Bohr effect）**：若pH↓或PCO₂↑，预期曲线右移，促进组织O₂释放。
   - **温度变化**：体温↑ → 曲线右移；体温↓ → 曲线左移，注意低温下血液呈鲜红色但组织可能缺氧。
   - **2,3-DPG水平**：慢性缺氧、贫血或高原适应时2,3-DPG↑ → 曲线右移；库存血因2,3-DPG↓ → 曲线左移，输注后O₂释放受限。
   - **CO中毒**：即使低浓度CO也可使曲线左移，同时形成樱桃红色HbCO，阻碍O₂结合与释放。

3. **判断临床干预方向**：
   - 曲线右移且过度（如严重酸中毒）→ 纠正原发病因（如补碱、改善通气）。
   - 曲线左移导致组织缺氧（如输注陈旧血、CO中毒）→ 考虑高压氧疗、避免使用低2,3-DPG血液制品。

4. **特殊人群注意**：
   - 胎儿Hb亲和力高（曲线左移），利于胎盘摄取O₂。
   - 镰状细胞贫血患者Hb亲和力降低，运氧能力受损。

> 提示：本技能仅适用于含可变构象Hb的全血、红细胞或Hb分子，不适用于无Hb溶液或固定构象突变体。
