# 计算食物特殊动力作用的额外能量消耗

## 何时使用
- 已知摄入食物的能量含量（单位：kJ）
- 食物成分为纯蛋白质、糖类、脂肪或混合膳
- 对象为健康成人或处于餐后安静状态的个体
- 不适用于严重代谢疾病患者

## 执行步骤
1. 确定摄入食物的主要营养成分类型（蛋白质、糖类、脂肪或混合膳）。
2. 获取该营养成分对应的SDA百分比：
   - 蛋白质：30%
   - 糖类：6%
   - 脂肪：4%
   - 混合膳：10%
3. 使用公式计算额外耗能：
   ```
   额外耗能（kJ） = 摄入该营养物质的能量（kJ） × SDA百分比
   ```
4. 将结果用于调整每日总能量需求评估。

> 注意：此计算反映长期能量调节机制，不用于评估即时饱腹感。
