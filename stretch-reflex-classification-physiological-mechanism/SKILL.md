# 牵张反射类型与生理机制

## 适用对象
- 具有完整神经支配的骨骼肌
- 人类及哺乳动物脊髓

## 前提条件
确保以下结构与功能正常：
- 肌梭结构完整
- α和γ运动神经元功能正常
- 脊髓反射弧未中断

## 执行步骤

1. **识别感受器与传入通路**
   - 确认肌梭为牵张反射的感受器，位于肌纤维间，含6–12根核内肌纤维。
   - 识别传入纤维类型：Ia类（快速动态响应）和II类（静态长度感知），均终止于α运动神经元。
   - 注意γ运动神经元调节肌梭敏感性，通过支配核内肌两端收缩成分实现。

2. **区分牵张反射类型**
   - **腱反射**：由快速牵拉肌腱触发（如叩击股四头肌肌腱），表现为快肌纤维同步收缩，动作明显；属单突触反射，中枢传导时间约0.7ms。
   - **肌紧张**：由缓慢持续牵拉引发（如直立时重力作用），表现为慢肌纤维交替轻度收缩，无明显动作；属多突触反射，是维持姿势的基础。

3. **分析α-γ共同激活机制**
   - 高位中枢同时激活α和γ运动神经元，确保肌肉收缩过程中肌梭仍能感知长度变化，维持反馈灵敏度。

4. **评估生理意义与临床应用**
   - 利用腱反射评估神经系统完整性：反射减弱提示反射弧损伤，亢进提示高位中枢（如锥体束）病变。
   - 解释肌紧张在对抗重力、维持直立姿势（如下肢伸肌、竖脊肌）中的作用。

5. **识别保护性抑制机制**
   - 当肌肉张力过大时，腱器官通过Ib传入纤维抑制α运动神经元，触发反牵张反射，防止肌肉或肌腱损伤。

## 使用时机
- 解读临床神经检查结果（如膝反射、踝反射）
- 分析姿势控制障碍的生理根源
- 区分上运动神经元与下运动神经元病变
- 理解运动协调中感觉反馈的作用
