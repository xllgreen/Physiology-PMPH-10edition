# 微循环三条血流通路的功能与调控

根据组织代谢需求或体温调节需要，选择性激活以下三条微循环通路之一或组合：

## 1. 迂回通路（营养通路）
- **适用场景**：组织需高效进行氧气、营养物质和代谢废物交换时。
- **执行步骤**：
  1. 确认血流路径为：微动脉 → 后微动脉 → 毛细血管前括约肌 → 真毛细血管网 → 微静脉。
  2. 判断毛细血管前括约肌处于舒张状态（受局部代谢产物如CO₂、H⁺、腺苷等调控）。
  3. 预期约20%真毛细血管在安静状态下开放，代谢活跃时比例上升。

## 2. 直捷通路（thoroughfare channel）
- **适用场景**：需快速维持静脉回心血量，同时允许少量物质交换（如骨骼肌活动期间）。
- **执行步骤**：
  1. 确认血流路径为：微动脉 → 后微动脉 → 通血毛细血管 → 微静脉。
  2. 注意该通路通常持续开放，血流速度快、阻力小。

## 3. 动静脉短路（非营养通路）
- **适用场景**：环境温度升高需散热，或病理状态下（如感染性休克）出现异常开放。
- **执行步骤**：
  1. 确认血流路径为：微动脉 → 动-静脉吻合支 → 微静脉。
  2. 判断是否因高温刺激导致交感神经抑制，使吻合支开放以增加皮肤血流量。
  3. 警惕“暖休克”风险：皮肤温暖但组织缺氧，因血液绕过真毛细血管。

> **注意**：动静脉短路在正常体温下通常关闭；仅在体温调节或特定病理条件下激活。
