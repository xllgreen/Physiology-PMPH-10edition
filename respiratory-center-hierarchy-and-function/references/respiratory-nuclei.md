# 呼吸中枢关键核团

## 延髓
- **背侧呼吸组（DRG）**：位于孤束核腹外侧，主要为吸气神经元
- **腹侧呼吸组（VRG）**：包括后疑核（吸气）、疑核（呼气）等
- **前包钦格复合体（Pre-Bötzinger Complex）**：节律起搏点

## 脑桥
- **臂旁内侧核（PB）与KF核（PBKF）**：合称呼吸调整中枢（PC），抑制吸气

## Lumsden横切实验总结
| 横切平面 | 效应 |
|----------|------|
| A（中脑-脑桥间） | 呼吸节律不变 |
| B（脑桥上-中部） | 呼吸变慢变深；+迷走切断 → 长吸式呼吸 |
| C（延髓-脑桥间） | 不规则喘息样呼吸 |
| D（脊髓-延髓间） | 呼吸停止 |
