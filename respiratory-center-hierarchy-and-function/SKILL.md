# 呼吸中枢层级结构与功能解析

## 何时使用
- 解释呼吸节律如何产生（如延髓Pre-Böt C的作用）
- 分析脑干横切实验结果（如Lumsden实验）
- 理解高位脑损伤或脊髓病变对呼吸的影响（如清醒可呼吸、入睡停止）

## 如何执行
1. 识别涉及的神经结构层级：
   - **脊髓**：运动神经元中继站，无节律生成功能
   - **延髓**：基本节律中枢（DRG主吸气，VRG含吸/呼神经元）
   - **脑桥**：PC（PBKF核）限制吸气，促进吸-呼转换
   - **高位脑**：实现随意控制（说话、屏气）
2. 结合实验证据判断功能：
   - B平面横切 + 迷走切断 → 长吸式呼吸（apneusis）
   - D平面横切 → 呼吸立即停止
3. 应用临床逻辑：
   - 自主呼吸依赖低位脑干
   - 随意呼吸依赖皮层-脊髓通路
   - 两者通路分离可导致特定病理状态

> 注意：完整呼吸需各级中枢协同，但节律核心位于延髓。

详细神经核团定位见 `references/respiratory-nuclei.md`。
