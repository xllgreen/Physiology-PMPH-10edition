# 自主性体温调节的负反馈机制

当内/外环境温度变化导致体温偏离下丘脑调定点（通常为37℃）时，执行以下负反馈调节流程：

## 1. 温度检测
- **外周感受器**：位于皮肤、黏膜、内脏，对温度变化速率敏感。
- **中枢感受器**：主要位于下丘脑PO/AH区，可响应0.1℃变化且无适应现象。

## 2. 信息整合
- 将感受器信号传入下丘脑体温调节中枢进行整合。

## 3. 效应器调控
- **若体温低于调定点**：
  1. 激活骨骼肌战栗产热。
  2. 收缩皮肤血管以减少散热。
- **若体温高于调定点**：
  1. 刺激汗腺分泌汗液。
  2. 扩张皮肤血管以增加散热。

## 4. 反馈校正
- 效应器活动使体温回归调定点，负反馈完成。
- **前馈机制提示**：在预期温度变化前（如运动前），机体可提前启动调节以减小波动。

> **关键约束**：当中枢或外周温度感受器受损，或发生非调节性体温升高（如中暑）时，此机制失效。
