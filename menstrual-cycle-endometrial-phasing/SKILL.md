# 月经周期子宫内膜分期判断

## 何时使用
- 用户提供月经周期第几天（以月经来潮第一天为D1）
- 用户提供雌激素或孕激素水平数据
- 用户提供超声内膜厚度或形态描述（如“三线征”）
- 需评估是否处于适合胚胎着床的窗口期
- 需鉴别异常子宫出血的可能内分泌原因

> **注意**：本技能不适用于妊娠期、绝经后或无排卵性月经患者。

## 执行步骤

### 步骤1：确认适用条件
- 确认对象为育龄期女性
- 确认存在规律月经来潮
- 排除妊娠、绝经或无排卵状态

### 步骤2：根据输入信息判断分期

#### 情况A：已知周期天数
- **D1–D5** → 月经期
- **D6–D14** → 增生期
- **D15–D28** → 分泌期（通常持续约14天）

#### 情况B：已知激素水平
- **雌激素主导上升，孕激素低** → 增生期
- **孕激素显著升高（黄体期水平）** → 分泌期
- **雌、孕激素均骤降** → 月经期

#### 情况C：已知超声内膜特征
- **内膜薄（<5mm），结构不清** → 月经期末或增生早期
- **内膜增厚（8–10mm），见“三线征”** → 增生晚期（接近排卵）
- **内膜厚、回声均匀、无三线征** → 分泌期

### 步骤3：输出分期结论及临床意义
- 明确指出当前内膜分期
- 关联对应的卵巢阶段与激素状态
- 提供临床应用建议（如：是否适合胚胎移植、是否提示无排卵等）

## 关键变量说明
- **周期天数**：以月经第一天为D1
- **内膜厚度**：超声测量的双层厚度（单位：mm）
- **雌激素水平**：主导增生期变化
- **孕激素水平**：主导分泌期变化，其升高标志已排卵
