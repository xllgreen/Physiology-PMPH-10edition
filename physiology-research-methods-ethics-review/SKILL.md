# 生理学研究方法选择与伦理审查

## 触发条件
开展生理功能机制研究时，必须根据研究对象选择合适方法并严格遵守3R原则（Reduction, Replacement, Refinement）。

## 执行流程

### 1. 判断研究对象类型
- **动物实验**：适用于机制探索，但需遵循伦理约束。
- **人体研究**：仅限无创或低风险方法，受严格伦理限制。

### 2. 动物实验方法选择
**急性实验**（短期、常具损伤性）：
- 在体（in vivo）：在麻醉或清醒动物身上操作，环境可控。
- 离体（in vitro）：分离器官或细胞后在人工环境中研究，排除干扰但生态效度较低。

**慢性实验**（长期、清醒状态下重复观察）：
- 需预处理（如造瘘、腺体摘除），待动物恢复后再实验。
- 结果更接近自然生理状态，适合行为与整体功能研究，但耗时长、变量多。

### 3. 人体生理研究方法
- 优先采用无创技术：心电图（ECG）、脑电图（EEG）、超声、CT、功能性磁共振成像（fMRI）等。
- 可结合大数据挖掘、基因组学开展个体化研究。
- 禁止实施任何有害或潜在损害性实验。

### 4. 伦理与方法论原则
- **严格执行3R原则**：
  - **Reduction（减少）**：最小化动物使用数量；
  - **Replacement（替代）**：尽可能使用非动物模型（如细胞培养、计算机模拟）；
  - **Refinement（优化）**：减轻动物痛苦，提升福利。
- 动物选择需考虑种属、性别、年龄、体重（例如猫对呕吐敏感，适用于相关研究）。
- **严禁将动物数据直接外推至人类**，因高级生理功能存在质的差异。

### 5. 科学思维流程
- 提出问题 → 建立假设 → 设计实验 → 实施并记录 → 分析讨论 → 得出结论。
- 整合信息论、控制论、系统论视角，推动向整合生理学与转化医学发展。

## 输出提醒
始终在方法选择中平衡科学性、伦理性与生态效度，避免不当外推或违反伦理规范。
