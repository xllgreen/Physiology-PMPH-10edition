# 排尿反射的神经调控与正反馈机制

## 何时使用
- 解释正常成人排尿的神经生理过程
- 分析因神经损伤（如脊髓病变）导致的排尿功能障碍
- 理解小儿遗尿或意识性排尿控制的生理基础
- 判断排尿反射是否可被触发（需满足膀胱容量与压力阈值）

## 执行步骤

1. **确认触发条件**：
   - 膀胱尿量达 400–500 ml
   - 膀胱内压 ≥ 10 cmH₂O
   - 牵张感受器激活且神经通路完整

2. **传入信号传导**：
   - 膀胱壁牵张感受器兴奋 → 冲动经盆神经传入 → 骶髓排尿初级中枢
   - 同时上传至脑干和大脑皮层，产生排尿欲

3. **传出神经反应**：
   - **副交感神经（盆神经）兴奋**：释放乙酰胆碱 → 激活逼尿肌 M 受体 → 逼尿肌收缩 + 尿道内括约肌舒张
   - **交感神经（腹下神经）抑制**：减少去甲肾上腺素释放 → 解除对逼尿肌的抑制及内括约肌收缩
   - **阴部神经抑制**：尿道外括约肌（骨骼肌）舒张

4. **启动正反馈循环**：
   - 尿液进入后尿道 → 刺激后尿道感受器 → 冲动再次传入脊髓 → 进一步增强排尿中枢活动 → 更强逼尿肌收缩 + 外括约肌持续开放
   - 循环持续直至膀胱排空

5. **辅助排尿机制**：
   - 腹肌与膈肌强力收缩 → 增加腹内压 → 协助尿液排出
   - 排尿末期：男性依赖球海绵体肌收缩清除尿道残余尿；女性主要依赖重力

6. **高级中枢调控**：
   - 大脑皮层可有意识抑制或促进排尿
   - 小儿因皮层发育未完善，控制能力弱，易发生遗尿

## 使用限制
- 不适用于脊髓休克期患者
- 不适用于传入或传出神经损伤者
- 若膀胱容量或压力未达阈值，反射不启动
