# TRP通道介导的皮肤温度觉分子机制

## 何时使用
- 用户询问皮肤对冷/热刺激的分子感知机制；
- 需要根据具体温度（如35℃、15℃、48℃）判断激活的TRP通道及产生的感觉类型；
- 分析为何超过46℃时热觉转为痛觉；
- 解释薄荷醇或辣椒素如何模拟温度觉。

## 执行步骤

1. **确认适用条件**：
   - 刺激部位为皮肤（不适用于内脏）；
   - 温度变化在TRP通道激活范围内；
   - TRP通道正常表达。

2. **判断温度区间并匹配TRP通道**：
   - 若温度在 **10–26℃** 范围内 → 激活 **TRPM8**（冷觉，可被薄荷醇激活）；
   - 若温度 **<17℃** → 可能激活 **TRPA1**（极冷觉）；
   - 若温度在 **27–34℃** → 激活 **TRPV4**（温和热觉）；
   - 若温度在 **33–40℃** → 激活 **TRPV3**（温暖觉）；
   - 若温度 **>42℃** → 激活 **TRPV1**（热觉/热痛，可被辣椒素激活）；
   - 若温度 **>40℃** → 可能激活 **TRPM3**；
   - 若温度 **>52℃** → 激活 **TRPV2**（强热痛）。

3. **判断感觉输出**：
   - **30–46℃**：热感受器放电频率随温度升高而增加 → 产生**热觉**；
   - **<30℃**：冷感受器放电增加 → 产生**冷觉**；
   - **>46℃**：热觉消失，激活伤害性感受器 → 转为**痛觉**。

4. **注意特殊现象**：
   - TRPA1与TRPV1常共定位，极冷刺激可能引发“热感”错觉；
   - 其他通道（如TREK/TRAAK、P2X3、ANO1）在特定温度下辅助换能，但核心温度觉由TRP通道主导。

5. **输出结果**：明确说明当前温度下激活的主要TRP通道、所属感觉类型（冷觉/热觉/痛觉）及是否涉及非温度配体（如辣椒素、薄荷醇）。
