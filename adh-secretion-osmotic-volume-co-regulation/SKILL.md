# ADH分泌的协同调节

## 何时使用
- 用户询问ADH分泌的调控机制
- 需要解释高渗性脱水、失血、大量饮水后的尿量变化
- 分析下丘脑-垂体轴在体液稳态中的作用
- 判断何种刺激（渗透压 vs 血容量）主导ADH释放

## 核心执行流程

### 1. 识别主要触发条件
- **血浆晶体渗透压升高1%～2%**（主要由Na⁺/Cl⁻引起）→ 强烈刺激ADH释放
- **循环血量或动脉血压下降≥5%～10%** → 激活容量/压力感受器通路，促进ADH释放

> 注意：葡萄糖或尿素引起的渗透压变化不触发ADH分泌。

### 2. 确定感受器通路
- **渗透压感受器**：位于下丘脑前部室周器，对Na⁺/Cl⁻敏感
- **容量感受器**：位于心房和大静脉，监测静脉回流
- **压力感受器**：位于颈动脉窦和主动脉弓，监测动脉血压（<100 mmHg时抑制减弱）

### 3. 应用协同调节规则
- 当**血容量/血压下降**时，**降低**渗透压触发ADH释放的阈值（提高敏感性）
- 当**血容量/血压升高**时，**升高**渗透压阈值（降低敏感性）

### 4. 推导生理结果
- ADH↑ → 肾集合管水重吸收↑ → 尿量↓、尿液浓缩
- ADH↓ → 水重吸收↓ → 水利尿（如饮清水后）

## 关键约束
- 需完整下丘脑-垂体轴及功能性感受器通路
- 慢性容量变化可能导致感受器敏感性适应性下调
