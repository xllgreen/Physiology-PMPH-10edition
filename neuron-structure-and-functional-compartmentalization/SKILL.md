# 神经元结构与功能分区

## 适用条件
- 对象必须是哺乳动物神经系统中的完整神经元
- 不适用于非神经元细胞或已发生变性的神经元

## 执行步骤

1. **识别神经元基本结构**：
   - **树突**：数量多、分支复杂，表面有树突棘，为主要突触后部位
   - **轴突**：通常仅一条，起始于轴丘；始段无髓鞘，负责动作电位起始
   - **轴突末段**：分枝形成神经末梢，含曲张体和突触囊泡

2. **划分功能区域**：
   - **胞体与树突**：接收并整合输入信号
   - **轴突始段**：整合信号并产生动作电位
   - **轴突主干**：传导动作电位
   - **突触末梢**：向下游神经元或效应细胞传递信息

3. **评估树突棘状态**：
   - 树突棘密度和形态反映脑可塑性
   - 发育期数量增加，与智力正相关
   - 智力障碍（如唐氏综合征）常表现为树突棘稀少且细长

4. **分类神经元类型**（按突起数量）：
   - 假单极：如背根神经节细胞
   - 双极：如视网膜双极细胞
   - 多极：如脊髓运动神经元、海马锥体细胞、小脑浦肯野细胞

## 关键变量
- **树突棘密度**：数值/形态指标，反映信息接收能力与可塑性
- **轴突长度**：解剖参数，投射神经元较长，中间神经元较短

## 输出用途
用于判断神经元类型、功能状态及发育或病理状况。
