# 中枢抑制详细机制

## 突触后抑制

### 侧支抑制（传入侧支抑制/交互抑制）
- **结构基础**：Ia类传入纤维分出侧支，兴奋α运动神经元的同时激活抑制性中间神经元（释放甘氨酸），后者抑制拮抗肌的α运动神经元。
- **生理意义**：确保主动肌收缩时拮抗肌舒张，实现精细运动协调。
- **示例**：膝跳反射中，股四头肌收缩的同时腘绳肌被抑制。

### 回返性抑制
- **结构基础**：α运动神经元轴突发出侧支，兴奋Renshaw细胞（脊髓内抑制性中间神经元），后者释放甘氨酸反馈抑制原α运动神经元及协同肌神经元。
- **功能**：
  - 防止运动过度
  - 限制放电频率
  - 同步运动神经元群
- **病理关联**：
  - 破伤风毒素：阻断甘氨酸释放 → 失去抑制 → 强直性痉挛
  - 士的宁（strychnine）：竞争性阻断甘氨酸受体 → 类似破伤风症状

## 突触前抑制

### 结构与离子机制
- **典型回路**：感觉末梢A → 投射神经元C；抑制性中间神经元B轴突终末包绕A末梢（轴-轴突触）。
- **GABA作用机制**：
  - A末梢内[Cl⁻]高 → E_Cl ≈ -25 mV（静息电位≈-70 mV）
  - GABA_A受体激活 → Cl⁻外流 → A末梢去极化（称为PAD, primary afferent depolarization）
  - PAD导致：
    - 动作电位幅度降低
    - 电压门控Ca²⁺通道开放减少
    - 递质释放量下降
- **其他机制**：
  - GABA_B受体激活 → G蛋白耦联 → K⁺通道开放 → 复极化加速 → Ca²⁺内流时间缩短
  - 自身受体负反馈：谷氨酸激活mGluR → 抑制Ca²⁺通道

### 功能意义
- 调节感觉信息传入强度（如痛觉闸门控制）
- 防止感觉过载
- 参与注意力调控

## 突触前易化（对比参考）
- **机制**：5-羟色胺 → 激活腺苷酸环化酶 → cAMP↑ → PKA磷酸化K⁺通道 → K⁺外流减少 → 动作电位时程延长 → Ca²⁺内流增加 → 递质释放增多
- **行为关联**：海兔缩鳃反射的敏感化（sensitization）

## 关键变量说明
- **Cl⁻平衡电位（E_Cl）**：决定GABA在末梢引起去极化（E_Cl > 静息电位）还是超极化（E_Cl < 静息电位）。在成熟中枢神经元胞体，E_Cl通常低于静息电位，故GABA引起超极化；但在初级传入末梢，因NKCC1转运体维持高[Cl⁻]_i，E_Cl高于静息电位，故GABA引起去极化。
- **递质类型**：
  - 突触后抑制：甘氨酸（脊髓）、GABA（脑干以上）
  - 突触前抑制：GABA（主要）、其他调质（如阿片肽）
