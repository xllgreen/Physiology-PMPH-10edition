# 基础代谢率计算与临床评估

## 何时使用
当已测定人类受试者或临床患者在基础状态（空腹、静卧、清醒、环境温度适宜）下的耗氧量和体表面积时，使用本技能计算BMR并判断其临床意义。

## 执行步骤

1. **确认前提条件**：
   - 已测定基础状态耗氧量（L/h）；
   - 已计算体表面积（m²）；
   - 已知对应年龄/性别的正常BMR平均值（参考标准表）。

2. **计算BMR绝对值**：
   - 使用公式：
     ```
     BMR [kJ/(m²·h)] = (20.20 × 耗氧量 [L/h]) ÷ 体表面积 [m²]
     ```

3. **计算相对偏差**：
   - 相对值 (%) = [(实测BMR − 正常平均值) ÷ 正常平均值] × 100%

4. **临床判断**：
   - **正常**：相对值在 ±15% 以内；
   - **异常提示**：相对值超过 ±20%，需进一步评估；
   - **典型病理关联**：
     - 甲状腺功能减退：BMR ↓20%～40%；
     - 甲状腺功能亢进：BMR ↑25%～80%；
     - 营养不良：BMR ↓20%～30%；
     - 发热：体温每升高1℃，BMR ↑约13%。

5. **输出结果**：
   - 返回BMR绝对值、相对值及是否超出正常范围的警示。

> 注：正常BMR参考值表、体表面积计算方法及非蛋白呼吸商说明详见 `references/bmr-reference-tables.md`。
