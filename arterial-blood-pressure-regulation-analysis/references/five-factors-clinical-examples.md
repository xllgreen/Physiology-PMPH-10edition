# 五大血压影响因素临床示例

## 每搏输出量变化
- **主动脉瓣关闭不全**：舒张期血液反流 → 有效搏出量↓ → 收缩压↓、舒张压↓↓ → 脉压↓
- **运动时**：心肌收缩力增强 → 搏出量↑ → 收缩压显著升高

## 心率影响
- **窦性心动过速**（如发热）：心率↑ → 舒张期缩短 → 舒张压升高为主
- **病态窦房结综合征**：心率过缓 → 舒张压下降更明显 → 脉压增大

## 外周阻力
- **原发性高血压早期**：小动脉痉挛 → 外周阻力↑ → 舒张压升高为主
- **感染性休克**：血管麻痹 → 外周阻力↓↓ → 舒张压显著下降

## 大动脉弹性
- **老年单纯收缩期高血压**：动脉硬化 → 收缩压↑、舒张压↓ → 脉压 > 60 mmHg

## 血量与容量失衡
- **脱水**：血量↓ → 平均充盈压↓ → 血压全面下降
- **过敏反应**：毛细血管通透性↑ + 血管扩张 → 有效循环血量↓ + 容量相对↑ → 严重低血压
