# 压力感受性反射详细机制

## 功能曲线特性
- S形曲线，中间陡峭段对应正常血压范围（约80–120 mmHg）
- 高血压时曲线右上移位（重调定/resetting），调定点升高至新稳态
- 最大敏感区间：传入冲动频率对血压变化最敏感的区域

## 传入神经差异
- 颈动脉窦：经舌咽神经（窦神经）传入，对血压变化更敏感
- 主动脉弓：经迷走神经传入（家兔中称为主动脉神经/减压神经）

## 中枢通路细节
- NTS → CVLM（抑制性）→ 抑制 RVLM → 降低交感输出
- NTS → 迷走神经背核/疑核 → 增强心迷走紧张

## 实验证据
- 切除缓冲神经后：血压波动幅度增大，但平均动脉压不变
- 证明其仅参与短期快速调节，不参与长期血压设定
