# 化学感受器信号转导机制

## 触发条件
当化学刺激物（气味分子或味觉物质）与特异性受体结合时，启动相应信号转导通路。

## 执行流程

### 嗅觉信号转导
1. 气味物质被嗅上皮黏液吸收并扩散至嗅毛。
2. 与嗅毛膜上的特异性气味受体（人类约1000种）结合。
3. 激活G蛋白 → 激活腺苷酸环化酶 → 生成cAMP（第二信使）。
4. cAMP打开化学门控钙通道 → Na⁺和Ca²⁺内流 → 嗅细胞去极化。
5. 去极化以电紧张方式扩布至轴突始段 → 产生动作电位 → 经嗅丝传至嗅球。

### 味觉信号转导（按基本味型）
- **咸味**：Na⁺经钠通道进入味细胞 → 去极化（可被阿米洛利阻断）。
- **酸味**：H⁺通过TRPP3通道进入细胞 → 去极化。
- **甜味、苦味、鲜味**：分别由T1R2/T1R3、T2R、T1R1/T1R3受体介导，均通过G蛋白 → 磷脂酶C → IP₃ → 内质网释放Ca²⁺ → 激活TRPM5通道 → 去极化。

### 共同特性
- 所有通路最终导致感受器细胞去极化，产生感受器电位。
- 味觉在20～30℃最敏感，且为快适应感受器。

## 使用前提
- 存在适宜化学刺激（气味或味觉物质）。
- 感受器细胞膜完整。
- 信号转导通路功能正常。

## 限制条件
- 不适用于非化学性感觉（如触觉、听觉）。
- 受损神经元无法正确转导信号。
