# 触-压觉感受器类型与Piezo通道机制分析

## 何时使用
- 用户研究皮肤或深部组织对机械力（拉伸、挤压、剪切）的感知机制；
- 需要区分快适应与慢适应机械感受器的分布及功能；
- 分析Piezo1或Piezo2在特定细胞（如默克尔细胞、背根神经节神经元）中的作用；
- 排除温度或化学刺激主导的感觉通路。

## 执行步骤

1. **确定皮肤区域类型**：
   - 若为**无毛皮肤**（如指尖、唇部），考虑四类主要机械感受器：环层小体、麦斯纳小体、鲁菲尼小体、默克尔盘；
   - 若为**有毛皮肤**，毛囊感受器替代麦斯纳小体，其余三类结构类似。

2. **判断感受器适应性**：
   - **快适应**：环层小体（深层压力/振动）、麦斯纳小体（轻触/低频振动）；
   - **慢适应**：鲁菲尼小体（皮肤拉伸）、默克尔盘（持续轻触与纹理分辨）。

3. **确认Piezo通道表达与功能**：
   - 检查目标细胞是否表达**Piezo2**（高表达于背根神经节神经元和默克尔细胞，介导轻触觉）；
   - 若涉及多种机械应力（如血管内皮、膀胱等非皮肤组织），考虑**Piezo1**的作用。

4. **验证前提条件**：
   - 确保存在机械刺激；
   - 确认Piezo1或Piezo2基因在相关细胞中表达。

5. **输出结果**：
   - 根据刺激性质、皮肤区域和细胞类型，明确参与的机械感受器及主导的Piezo通道亚型。

> 注意：本技能仅适用于哺乳动物躯体机械感知，不适用于听觉、平衡觉或内脏机械感知等其他机械感觉系统。
