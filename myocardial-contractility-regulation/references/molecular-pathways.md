# 心肌收缩能力调控的分子通路细节

## 1. 儿茶酚胺-cAMP-PKA通路
- 去甲肾上腺素/肾上腺素 → β₁-肾上腺素受体 → Gs蛋白 → 腺苷酸环化酶激活 → cAMP↑ → PKA激活
- PKA磷酸化：
  - L型钙通道（增加开放概率，Ca²⁺内流↑）
  - 磷脂抑制蛋白（PLB）→ 解除对SERCA的抑制 → 肌浆网Ca²⁺回收↑（加速舒张）
  - 肌钙蛋白I（降低Ca²⁺亲和力，促进舒张）

## 2. 牵张激活通路
- 心肌细胞牵张 → Piezo1通道开放 → Ca²⁺内流 + ROS生成↑
- 同时诱导AngⅡ自分泌 → AT1受体 → NADPH氧化酶 → ROS↑ → RyR2敏感性↑ → Ca²⁺瞬变幅度↑

## 3. 甲状腺激素作用机制
- T3/T4进入心肌细胞 → 核受体结合 → 上调：
  - α-肌球蛋白重链（高ATP酶活性）
  - β₁-肾上腺素受体密度
  - Na⁺-K⁺-ATP酶表达
- 甲减时：α-MHC↓、β-MHC↑ → 收缩速度↓、ATP利用效率↓

## 4. 钙增敏剂作用
- 如左西孟旦：结合肌钙蛋白C，增强其对Ca²⁺亲和力，不增加胞内Ca²⁺浓度，减少心律失常风险

## 5. 病理限制条件
- 在心肌缺血、ATP耗竭或严重酸中毒时，即使Ca²⁺升高，横桥循环仍受阻，收缩能力无法提升
