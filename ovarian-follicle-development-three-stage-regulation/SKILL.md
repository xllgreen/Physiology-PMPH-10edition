# 卵泡发育三阶段调控机制

## 适用场景
- 评估育龄期女性卵巢储备与功能
- 解读超声监测下的卵泡生长数据
- 分析多囊卵巢综合征（PCOS）、卵巢早衰等疾病的病理机制
- 指导人工促排卵方案设计

> **触发条件**：仅适用于青春期至绝经前、卵巢储备未耗竭、下丘脑-垂体-卵巢轴功能正常的女性。

## 执行步骤

### 1. 判断当前是否满足应用前提
- 确认对象为育龄期女性（非绝经后）
- 排除卵巢早衰或无原始卵泡个体
- 确保下丘脑-垂体-卵巢轴结构与功能完整

### 2. 根据卵泡直径与周期天数定位发育阶段

#### 阶段一：FSH非依赖的缓慢生长阶段
- **起始形态**：原始卵泡（直径≈50μm，单层梭形前颗粒细胞 + 初级卵母细胞停滞于减数分裂前期）
- **演进路径**：原始卵泡 → 初级卵泡（立方状多层颗粒细胞 + 透明带形成） → 次级卵泡（窦前卵泡）
- **关键特征**：
  - 完全不依赖垂体FSH
  - 生长极其缓慢（可持续十余年）
  - 由卵巢局部旁分泌因子驱动
- **何时使用此阶段判断**：当分析胎儿期至成年早期的卵泡池建立，或解释为何部分卵泡可长期静止

#### 阶段二：FSH反应性生长阶段
- **起始点**：窦前卵泡
- **驱动因素**：垂体基础FSH分泌（非周期性波动）
- **时间跨度**：约75～85天
- **终点形态**：直径2～5mm的小窦状卵泡
- **关键特征**：需最低阈值FSH支持，但不受月经周期FSH峰影响
- **何时使用此阶段判断**：当解读卵泡从静止池进入可募集池的过渡过程，或评估基础FSH水平对卵泡启动的影响

#### 阶段三：FSH高度依赖的快速生长阶段（周期募集）
- **触发时机**：月经周期黄体期末至卵泡早期（约周期第1～5天），FSH短暂升高
- **过程**：10～20个小窦状卵泡被同步募集进入快速生长
- **选择机制**：
  - 卵泡分泌雌激素与抑制素 → 负反馈抑制FSH → FSH浓度下降
  - 仅FSH阈值最低、生长最快者成为优势卵泡（通常在周期第5～7天确定）
  - 其余卵泡因FSH不足而闭锁
- **临床关联**：
  - 选择失败 → 多卵泡发育 → 多胎妊娠风险
  - PCOS：募集过多但无优势化 → 无排卵
  - 促排卵治疗：外源FSH维持高于闭锁阈值 → 多个卵泡成熟
- **何时使用此阶段判断**：当分析周期性排卵机制、解释超声监测中卵泡选择现象，或设计/调整促排卵方案

### 3. 输出结果
根据输入的卵泡直径、周期天数及激素背景，明确指出：
- 当前所处发育阶段
- 主导调控激素（是否依赖FSH）
- 是否符合正常选择逻辑
- 潜在临床意义（如闭锁风险、优势化可能性）
