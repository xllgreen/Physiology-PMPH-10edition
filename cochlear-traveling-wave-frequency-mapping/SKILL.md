# 基底膜行波频率定位机制

## 何时使用
- 用户询问声音频率如何在耳蜗中被空间编码
- 需要解释高频/低频听力损失与耳蜗损伤部位的对应关系
- 涉及听觉生理学中行波理论的基本原理
- 分析毛细胞激活与听神经放电模式的机制

## 执行步骤

1. **确认前提条件**：确保场景满足以下条件：
   - 声波经中耳传入前庭窗
   - 耳蜗淋巴液不可压缩
   - 基底膜具有从底（硬）到顶（软）的弹性梯度

2. **启动行波传播**：
   - 声波振动经听骨链传至前庭窗，引起耳蜗内淋巴液压力变化
   - 行波从基底膜底部（靠近前庭窗）开始向蜗顶方向传播

3. **确定最大振幅位置**：
   - 高频声波（>2000 Hz）→ 最大振幅靠近蜗底
   - 中频声波 → 最大振幅位于基底膜中部
   - 低频声波（<500 Hz）→ 最大振幅靠近蜗顶

4. **激活毛细胞与神经编码**：
   - 位于最大振幅区的毛细胞受到最强机械刺激
   - 对应听神经纤维发放冲动最多，向听觉中枢传递特定音调信息

5. **临床关联判断**：
   - 若耳蜗底部损伤 → 预期高频听力下降
   - 若耳蜗顶部损伤 → 预期低频听力下降

> 注意：本机制不适用于骨传导主导的听觉路径或耳蜗结构严重损伤的情况。
