# 原发性主动转运

## 何时使用
当需解释或模拟细胞如何逆浓度梯度转运带电离子以维持膜电位、细胞容积或为其他过程供能时使用本技能。

**触发条件**：存在ATP、特异性离子泵，且需逆电化学梯度转运离子。

## 执行步骤
1. **识别目标离子与对应泵**：
   - Na⁺/K⁺ → Na⁺,K⁺-ATP酶
   - Ca²⁺ → PMCA（质膜）或 SERCA（内质网）
   - H⁺ → H⁺,K⁺-ATP酶（胃壁细胞）
2. **启动ATP水解循环**：
   - ATP结合并水解，使泵磷酸化
   - 构象从E1转为E2，亲和力反转
   - 离子释放至目标侧
   - 去磷酸化恢复原始构象
3. **确认转运比例**：
   - Na⁺,K⁺泵：3 Na⁺出 / 2 K⁺入 / 1 ATP
   - PMCA：1 Ca²⁺出 / 1 ATP
   - SERCA：2 Ca²⁺入内质网 / 1 ATP
   - H⁺,K⁺泵：1 H⁺出 / 1 K⁺入 / 1 ATP
4. **评估调控因素**：
   - 激活：胞内高Na⁺、胞外高K⁺、β受体激动剂
   - 抑制：ATP缺乏、低温、酸中毒

## 输出结果
建立并维持跨膜离子浓度梯度（如胞外[Na⁺]为胞内10倍），产生生电效应，为细胞电活动、容积稳定及继发性转运提供基础。
