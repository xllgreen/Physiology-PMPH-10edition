# Frank-Starling机制与后负荷调节分析

## 适用条件
- 对象：健康左心室、右心室或整体心血管系统
- 前提：心肌结构完整，前负荷在生理范围（EDP 5–20 mmHg），无严重心衰或心律失常
- 禁忌：严重心力衰竭、心肌纤维化、急性心肌梗死或心室颤动

## 执行步骤

### 1. 前负荷变化分析（Frank-Starling机制）
- 若静脉回心血量增加 → 心室舒张期末容积（EDV）↑ → 心肌初长度↑ → 收缩力↑ → 搏出量↑
- 根据EDP判断调节有效性：
  - EDP 5–15 mmHg：调节敏感，每搏功显著增加
  - EDP 15–20 mmHg：调节趋于饱和
  - EDP >20 mmHg：正常心肌无明显降支，但调节失效

### 2. 后负荷变化分析
- 若大动脉血压升高：
  - 等容收缩期延长，射血期缩短
  - 射血速度减慢，搏出量暂时减少
- 若大动脉血压降低：有利于射血，搏出量增加

### 3. 继发代偿评估
- 后负荷升高导致搏出量↓ → 心室收缩期末容积↑ → 若静脉回流不变，则EDV↑
- 触发Frank-Starling机制 → 收缩力增强 → 搏出量回升，EDV逐渐恢复
- 在80–170 mmHg主动脉压范围内，心输出量通常保持稳定

### 4. 输出结果
- 预测搏出量变化趋势
- 判断是否处于有效调节范围
- 说明代偿机制是否启动及最终稳态效果
