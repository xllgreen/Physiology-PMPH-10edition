# 营养性贫血的营养物质评估与诊断

## 何时使用
当患者出现贫血相关症状（如乏力、苍白、心悸）且临床怀疑由营养缺乏引起时，使用本技能评估红细胞生成所需的关键营养物质水平，并鉴别贫血类型。

## 执行步骤

1. **确认适用条件**：
   - 患者存在造血干细胞且骨髓功能正常；
   - 排除非营养性贫血（如溶血性、再生障碍性贫血）。

2. **识别关键营养物质及其作用**：
   - **铁与蛋白质**：用于血红蛋白合成；
   - **叶酸与维生素B12**：作为DNA合成辅酶，促进红细胞核成熟；
   - 其他辅助营养素包括维生素B2、B6、C、E及铜、锰、钴、锌等。

3. **评估缺乏后果**：
   - 缺铁 → 小细胞低色素性贫血（缺铁性贫血）；
   - 缺叶酸或维生素B12 → DNA合成障碍 → 巨幼细胞贫血。

4. **分析吸收与储存特点**：
   - 维生素B12需内因子结合并在回肠远端吸收，胃部手术或自身免疫可致吸收障碍；
   - B12体内储存量大（4–5 mg），缺乏后3–5年才发病；
   - 叶酸储存较少（5–20 mg），摄入不足3–4个月即可致病。

5. **输出诊断结论**：
   - 根据缺乏的营养物质类型，明确贫血分类，指导后续补充治疗。
