# 视紫红质光化学循环机制

## 何时使用
- 分析视杆细胞感光换能的分子机制
- 解释夜盲症（nyctalopia）的生化基础
- 阐述维生素A在视觉循环中的作用
- 区分视杆与视锥细胞感光色素的差异（仅限视杆系统）

## 执行步骤

1. **初始状态（暗处）**：确认视紫红质由视蛋白与11-顺视黄醛共价结合形成，位于视杆细胞外段盘膜。

2. **光照触发分解**：
   - 光照使11-顺视黄醛异构化为全反型视黄醛；
   - 导致视紫红质构象改变（“漂白”）并与视蛋白分离；
   - 游离视蛋白激活转导蛋白（Gt），启动信号级联。

3. **暗处再合成路径**：
   - 全反型视黄醛从视杆细胞释放，被视网膜色素上皮细胞（RPE）摄取；
   - RPE中的异构酶将其转化为11-顺视黄醛；
   - 11-顺视黄醛返回视杆细胞，与视蛋白重新结合再生视紫红质。

4. **替代再合成路径（较慢）**：
   - 全反型视黄醛先还原为全反型视黄醇（维生素A）；
   - 经异构酶作用转为11-顺视黄醇；
   - 最终氧化为11-顺视黄醛参与合成。

5. **评估功能状态**：
   - 视紫红质浓度高 → 支持暗视觉；
   - 浓度低（如维生素A缺乏）→ 视杆系统失活，引发夜盲。

> 注意：本机制不适用于视锥细胞感光色素；再合成必须依赖视网膜色素上皮中的异构酶及维生素A储备。
