# 氧解离曲线三段功能解析

## 适用场景
- 分析动脉血（PO₂≈100 mmHg）与静脉血（PO₂≈40 mmHg）之间的氧释放量
- 评估安静状态下组织的正常供氧能力
- 判断运动时肌肉等高代谢组织如何动员氧储备
- 解释为何PO₂≥60 mmHg时不易出现明显低氧血症

## 执行步骤
1. **确认测定条件**：确保分析基于标准条件（pH 7.4、PCO₂=40 mmHg、温度37°C、Hb=15 g/100 ml）。若存在异常血红蛋白（如胎儿Hb、镰状细胞Hb）或极端pH/温度，本技能不适用。
2. **确定PO₂范围**：根据目标部位（动脉血、静脉血、组织毛细血管、运动肌肉）判断当前PO₂所处曲线段：
   - 上段：PO₂ 60–100 mmHg
   - 中段：PO₂ 40–60 mmHg
   - 下段：PO₂ 15–40 mmHg
3. **应用对应生理逻辑**：
   - **上段**：曲线平坦，PO₂变化对氧饱和度影响小，保障基础氧合安全。
   - **中段**：曲线较陡，反映安静状态下组织的常规氧释放（约5 ml O₂/100 ml血液）。
   - **下段**：曲线最陡，支持运动等高需求状态下的氧储备动员（最多释放15 ml O₂/100 ml血液）。
4. **输出判断结果**：根据PO₂所处区段，明确当前属于“安全供氧区”、“正常供氧区”还是“储备动员区”。

## 注意事项
- 仅适用于成人正常血红蛋白及标准生理条件。
- 运动状态下组织PO₂可低至15 mmHg，此时应使用下段逻辑。
- 动脉血PO₂低于60 mmHg时已脱离上段保护范围，需警惕低氧血症。
