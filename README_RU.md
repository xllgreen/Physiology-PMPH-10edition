## О проекте

Проект систематизирует фундаментальные знания по физиологии человека по **12 основным категориям**, охватывая **165 ключевых навыков**.

**Целевая аудитория**: Студенты-медики, клиницисты, преподаватели физиологии, исследователи

**Учебник**: *Физиология*, 10-е издание (PMPH)
