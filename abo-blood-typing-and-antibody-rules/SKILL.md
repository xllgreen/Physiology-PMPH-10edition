# ABO血型分型与抗体分布规则

## 何时使用
- 需要确定个体的ABO血型表现型（A、B、AB或O）
- 进行输血前交叉配型
- 解释正向定型（红细胞抗原检测）与反向定型（血清抗体检测）结果不一致的情况
- 怀疑存在A亚型（如A₂、Aₓ）导致误判

> **注意**：本技能不适用于孟买型等罕见非ABO血型系统；新生儿仅适用正向定型（因其抗体来自母体）。

## 执行步骤

### 1. 正向定型：根据红细胞抗原确定初步血型
检查红细胞表面抗原类型：
- 仅有A抗原 → 初步判定为A型
- 仅有B抗原 → 初步判定为B型
- 同时有A和B抗原 → 判定为AB型
- 无A、B抗原 → 判定为O型

### 2. 反向定型：用血清抗体验证血型
检查血清中天然抗体类型以确认结果：
- A型血清应含抗B抗体
- B型血清应含抗A抗体
- AB型血清应不含抗A、抗B抗体
- O型血清应含抗A和抗B两种抗体

若正反定型结果矛盾，需考虑亚型或技术误差。

### 3. 评估亚型干扰风险
- 若疑似A型但与B型血清凝集微弱或无凝集，考虑A₂或Aₓ亚型
- A₂型红细胞仅表达A抗原（无A₁抗原），其血清可能含抗A₁抗体
- Aₓ型红细胞与B型血清不凝集（或极弱），但可与O型血清凝集
- 对弱反应样本，加用O型血清测试以避免误判为O型

### 4. 输出结果
- 明确报告ABO血型表现型（A、B、AB或O）
- 若存在亚型可能性，提示“需进一步亚型鉴定”

> 如遇无法解释的凝集模式或罕见表型，请转交专业输血实验室处理。
