# 肺顺应性与弹性阻力评估

## 何时使用
- 需评估肺或胸廓在无气流状态下的扩张能力
- 需判断吸气是否费力（如肺纤维化、肺水肿患者）
- 需区分弹性阻力与非弹性阻力对通气的影响

## 执行步骤

### 1. 确认适用条件
- 处于静态状态（气流停止，如屏气）
- 可测量容积变化（ΔV）和跨壁压变化（ΔP）
- 对象为有腔弹性器官（如肺、胸廓）

### 2. 获取必要参数
- **ΔV**：腔内容积变化（单位：升，L）
- **ΔP**：跨壁压变化（单位：cmH₂O），即肺内压与胸膜腔内压之差
  - 屏气且气道通畅时，肺内压 = 大气压（设为 0），故 ΔP = -胸膜腔内压变化

### 3. 计算顺应性
- 使用公式：**C = ΔV / ΔP**
- 单位：L/cmH₂O

### 4. 解读结果
- **C 值大** → 弹性阻力小 → 易扩张 → 吸气省力（如肺气肿）
- **C 值小** → 弹性阻力大 → 难扩张 → 吸气费力（如肺纤维化）

> **注意**：此方法仅适用于静态顺应性评估；动态条件下需额外考虑气道阻力等非弹性因素。
