# 内源性与外源性凝血途径整合机制

## 触发条件
当血管损伤导致组织因子暴露或血液接触胶原等带负电荷表面时，启动凝血途径整合机制。

## 执行步骤
1. **确认前提条件**：
   - 血管内皮已损伤
   - 组织因子或胶原暴露
   - Ca²⁺存在

2. **启动外源性途径**：
   - 组织因子（TF）与FVIIa及Ca²⁺形成FVIIa-TF复合物
   - 该复合物高效激活FX → FXa，并同时激活FIX → FIXa

3. **启动内源性途径（表面激活）**：
   - FXII接触胶原活化为FXIIa
   - FXIIa激活FXI → FXIa（需高分子量激肽原辅助）
   - FXIa激活FIX → FIXa

4. **整合与放大**：
   - FIXa与FVIIIa在血小板膜上形成内源性tenase，大量激活FX → FXa
   - 外源性途径产生的初始FXa生成少量凝血酶，后者正反馈激活FXI、FV、FVII及血小板，显著放大反应
   - FVIIa-TF可直接激活FIX，绕过FXI进入内源性放大环路

5. **评估生理意义**：
   - 外源性途径启动快，负责初始触发
   - 内源性途径（尤其FIX-FVIII轴）负责信号放大和维持
   - FXI缺乏（血友病C）出血轻微，而FVIII/FIX缺乏（A/B型）出血严重，反映内源性放大环路的核心地位

6. **输出结果**：判断是否能协同生成足量FXa以驱动凝血酶爆发式产生；若关键节点缺陷，提示相应凝血障碍风险。

> 注意：FXI、前激肽释放酶或高分子量激肽原缺乏者出血轻微，说明经典内源性启动（FXII→FXI）非生理性止血所必需。
