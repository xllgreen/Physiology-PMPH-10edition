# 化学突触的结构与分类

## 何时使用
- 用户询问神经元之间如何通过化学物质传递信息
- 需要区分定向突触与非定向突触的结构和功能差异
- 分析突触前/后结构、突触间隙宽度或递质释放模式
- 判断特定神经通路（如交感神经节后纤维）是否属于非定向突触

## 执行步骤

1. **确认适用条件**：
   - 确保涉及的是化学突触（存在神经递质释放，具备突触前与突触后成分）
   - 排除电突触或无递质参与的信号传递场景

2. **识别基本结构**：
   - 突触前膜（含突触囊泡）
   - 突触间隙（宽20～40 nm）
   - 突触后膜（厚约7.5 nm，含受体和信号蛋白）

3. **分类突触类型**：
   - **定向突触**：递质仅作用于突触后膜局限区域；具有活性区（active zone）和突触后致密区（PSD）；常见于神经肌肉接头、轴突-树突/胞体/轴突连接。
   - **非定向突触**：无固定突触后结构；递质从曲张体释放，扩散至>20 nm（可达400 nm以上）的效应细胞；效应取决于靶细胞是否表达相应受体（如肾上腺素能受体）；多见于单胺能神经元和交感神经节后纤维。

4. **关联囊泡类型与释放位置**：
   - 小而清亮囊泡（含乙酰胆碱或氨基酸类递质）→ 位于活性区
   - 小而致密中心囊泡（含儿茶酚胺类递质）→ 位于活性区
   - 大而致密中心囊泡（含神经肽类递质）→ 均匀分布于末梢内，可从膜任意部位释放

5. **输出结果**：明确突触类型、结构特征及其对信号传递范围和特异性的影响。
