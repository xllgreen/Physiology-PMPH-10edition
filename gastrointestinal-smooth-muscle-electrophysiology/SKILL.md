# 消化道平滑肌电生理与收缩机制分析

## 适用对象
- 胃肠道平滑肌细胞
- 肠神经系统调控下的消化道运动

## 前提条件
- 存在卡哈尔间质细胞（ICC）
- 细胞膜具备电压门控钙通道

## 使用限制
- 不适用于骨骼肌或心肌
- 离体实验需在适宜人工环境中维持功能

## 执行流程

1. **识别三种电活动形式**：确认是否存在静息电位（-60～-50 mV）、慢波电位（基本电节律，BER）和动作电位。

2. **评估慢波特性**：
   - 判断慢波是否起源于卡哈尔间质细胞（ICC），并通过缝隙连接扩布至平滑肌；
   - 根据部位确定慢波频率（胃约3次/分，十二指肠约12次/分，回肠末端8～9次/分）；
   - 测量慢波幅度（通常10～15 mV）和持续时间（数秒至十几秒）。

3. **判断收缩触发条件**：
   - 设定机械阈（mechanical threshold）和电阈（electrical threshold）；
   - 若慢波去极化 ≥ 机械阈，则引发小幅度收缩，其强度与慢波幅度正相关；
   - 若慢波去极化 ≥ 电阈，则触发动作用电位，动作电位数量越多，收缩越强。

4. **分析动作电位特征**：
   - 确认去极化由Ca²⁺内流驱动（非Na⁺），表现为上升支缓慢、持续时间长；
   - 复极化由K⁺外流完成；
   - Ca²⁺内流量显著高于仅达机械阈时，导致更强收缩。

5. **综合解释三联关系**：
   - 慢波作为“起步电位”，决定收缩的节律、方向和速度；
   - 动作电位叠加于慢波之上，调控收缩强度；
   - 收缩总是继动作电位之后发生；无动作电位时仅有微弱收缩。

## 关键变量
- `slow_wave_amplitude`（mV）：慢波电位幅度
- `mechanical_threshold`（mV）：引发机械收缩的膜电位阈值
- `electrical_threshold`（mV）：触发动作用电位的膜电位阈值
- `Ca2+_influx`：细胞内钙离子浓度变化
