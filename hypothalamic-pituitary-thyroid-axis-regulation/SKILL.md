# 下丘脑-垂体-甲状腺轴调控机制

## 何时使用
- 当分析血清TSH、T₃、T₄水平异常的原因时
- 在评估甲状腺功能亢进或减退的病理机制时
- 解释妊娠期垂体对缺氧敏感性及希恩综合征风险时
- 注意：不适用于格雷夫斯病等自身免疫性甲状腺疾病或碘阻滞效应主导的情况

## 执行步骤
1. **确认前提条件**：
   - 血液中存在游离甲状腺激素（TH）
   - 下丘脑释放激素（如TRH）处于基础分泌状态
   - 靶腺（甲状腺）功能完整，负反馈通路正常

2. **识别促激素结构特征**：
   - TSH为糖蛋白，由α和β亚单位组成异二聚体
   - α亚单位通用，β亚单位决定特异性；单独β亚单位无活性

3. **追踪负反馈调节路径**：
   - 血TH升高 → TH进入腺垂体TSH细胞 → 抑制TSH α/β亚单位基因转录
   - TH下调TSH细胞膜上TRH受体数量，降低对TRH敏感性
   - TH同时抑制下丘脑TRH前体原基因转录 → 减少TRH合成
   - 最终：TSH和TRH分泌减少 → 甲状腺TH合成下降 → 血TH回归稳态

4. **评估反向调节与病理关联**：
   - 长期低TH（如缺碘）→ 负反馈减弱 → TSH升高 → 甲状腺代偿性增生（地方性甲状腺肿）
   - 妊娠期垂体增生、耗氧增加 → 大出血可致缺血坏死（希恩综合征）→ 多腺体功能减退

5. **输出结果**：
   - 判断轴系功能是否正常
   - 提示是否存在垂体或下丘脑病变
   - 警惕多内分泌腺功能减退风险
