# 肺通气动力与胸膜腔负压机制

## 何时使用
- 需要解释外界气体如何进入肺泡并完成交换
- 需判断肺是否能随胸廓同步扩张（如怀疑气胸或胸壁损伤）
- 需分析静脉/淋巴回流受阻是否与胸腔负压异常相关

## 执行步骤

### 1. 确认前提条件
- 呼吸道通畅
- 胸膜腔密闭
- 中枢神经系统调控正常

### 2. 分析肺通气动力来源
- **吸气（主动过程）**：膈肌和肋间外肌收缩 → 胸廓容积增大 → 肺被动扩张 → 肺内压低于大气压 → 气体流入
- **平静呼气（被动过程）**：呼吸肌舒张 → 肺弹性回缩 → 胸廓缩小 → 肺内压高于大气压 → 气体流出
- **用力呼吸**：辅助吸气肌（如胸锁乳突肌）或呼气肌（如腹肌）参与，显著改变胸腔容积和压力

### 3. 评估胸膜腔负压作用
- 负压由肺弹性回缩力与胸廓扩展力对抗形成
- 维持肺紧贴胸廓，实现耦联运动
- 促进胸腔内静脉和淋巴管扩张，利于回流

### 4. 判断异常情况
- 若胸膜腔与大气相通（如气胸），负压消失 → 肺塌陷 + 回流受阻 → 需紧急干预

> **注意**：本技能不适用于气胸、严重胸壁损伤或呼吸肌麻痹等病理状态。
