# 去大脑僵直分析

## 何时使用
- 在麻醉动物（如猫）中进行中脑上、下丘之间切断术后出现四肢伸直、脊柱挺硬、头尾昂起、角弓反张。
- 在人类患者中观察到头后仰、四肢僵直伸直、上臂内旋、手指屈曲等体征，提示中脑病变。

## 执行步骤

1. **确认现象符合去大脑僵直定义**：
   - 四肢伸直坚硬如柱、脊柱强直、角弓反张。
   - 发生于中脑上、下丘之间脑干切断后。

2. **验证发生基础**：
   - 确认存在脊髓牵张反射。
   - 若局部麻醉或切断脊髓后根后僵直消失，则支持该机制。

3. **区分僵直类型**：
   - **若腰骶部后根完整**：
     - 切断后根后僵直消失 → 判断为 **γ僵直**。
     - 主要通路为网状脊髓束。
   - **若后根已切断**：
     - 切除小脑前叶蚓部后僵直再现 → 判断为 **α僵直**。
     - 再切断第Ⅷ对脑神经后僵直消失 → 确认前庭脊髓束为主要通路。

4. **临床解读**：
   - 人类出现去大脑僵直提示脑干严重受损，预后不良。
   - 类型判断有助于定位病变涉及的下行通路（网状脊髓束 vs 前庭脊髓束）。

> 注意：γ僵直依赖后根传入，α僵直可在后根切断后仍存在。
