# 毛细胞机械-电换能机制分析

## 适用对象
- 内毛细胞
- 外毛细胞
- 盖膜
- 基底膜
- 内淋巴

## 使用前提
确保以下条件成立：
- 基底膜与盖膜之间存在剪切运动
- 毛细胞纤毛顶部存在顶连（tip link）
- 内淋巴富含K⁺且电位为+80 mV
- 毛细胞静息电位为-80 ~ -70 mV

## 执行步骤

1. **确认剪切运动产生**
   - 验证基底膜振动是否导致其与盖膜发生相对上下位移
   - 判断外毛细胞较长纤毛是否因嵌入盖膜而弯曲，内毛细胞纤毛是否因内淋巴流动而弯曲

2. **检查纤毛束结构完整性**
   - 确认纤毛间由侧连（side link）连接成束，保证同步弯曲
   - 验证短纤毛顶端与相邻长纤毛之间存在顶连，连接机械门控阳离子通道

3. **分析通道开闭与电位变化**
   - 若基底膜上移 → 纤毛向长纤毛侧弯曲 → 顶连拉伸 → 通道开放 → K⁺大量内流 → 细胞去极化
   - 若基底膜下移 → 纤毛向短纤毛侧弯曲 → 顶连松弛 → 通道关闭 → K⁺内流终止 → 细胞超极化

4. **区分内、外毛细胞功能**
   - **内毛细胞**：去极化激活电压门控Ca²⁺通道 → Ca²⁺内流 → 谷氨酸释放 → 兴奋听神经传入纤维
   - **外毛细胞**：去极化引起胞体缩短，超极化引起伸长（由prestin蛋白介导）→ 增强基底膜振动 → 起“耳蜗放大器”作用

5. **评估关键离子驱动力**
   - 计算跨膜电位差（内淋巴+80 mV vs 细胞内-80 mV ≈ 150–160 mV）
   - 确认该高驱动力是K⁺快速内流、产生显著感受器电位的基础

6. **验证机制完整性**
   - 若prestin失活，则外毛细胞丧失电能动性与放大功能，可能导致听力下降

> **注意**：本机制不适用于纤毛结构破坏或内淋巴离子成分/电位异常的情况。
