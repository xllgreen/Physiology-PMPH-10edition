# 心室射血分数计算与临床解读

## 适用条件
- 对象：左心室、右心室、心力衰竭患者或需心脏功能评估者
- 前提：已通过超声心动图、MRI或心导管获得EDV和ESV数据
- 禁忌：严重心律失常、显著局部室壁运动异常或假性正常化舒张功能障碍

## 执行步骤

### 1. 计算射血分数（EF）
使用公式：
```
EF = [(EDV - ESV) / EDV] × 100%
```
其中：
- EDV：心室舒张期末容积（mL）
- ESV：心室收缩期末容积（mL）

### 2. 判断正常范围
- 左心室EF（LVEF）：55%–70%
- 右心室EF（RVEF）：45%–60%

### 3. 心力衰竭分类
- LVEF ≥50%：射血分数保留型心衰（HFpEF）
- LVEF 41%–49%：射血分数中间值型心衰（HFmrEF）
- LVEF ≤40%：射血分数降低型心衰（HFrEF）

### 4. 输出结果
- EF百分比数值
- 心功能状态分类
- 是否提示收缩功能障碍
- 注意事项（如EF不能反映舒张功能，运动员或妊娠者EF可能生理性偏高）
