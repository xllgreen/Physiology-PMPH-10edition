# 动脉血氧饱和度（SaO₂）临床监测

## 何时使用
- 患者为ICU住院者或接受呼吸功能监测
- 患者当前**未接受外源性氧气治疗**
- 指套式SaO₂读数异常（尤其<92%）或临床表现与SaO₂不符

## 执行步骤
1. **确认监测条件**：确保使用指套式光电传感器进行连续SaO₂监测，且患者处于未吸氧状态。
2. **读取SaO₂数值**并按以下标准处理：
   - **95%～98%**：属正常范围，继续常规监测。
   - **92%～94%**：提示轻度低氧风险，密切观察呼吸频率、节律、意识状态及皮肤黏膜颜色。
   - **<92%**：视为低氧血症警戒阈值，**立即执行动脉血气分析**以确认真实PaO₂和氧合状态。
3. **评估局限性**：若患者存在严重贫血、一氧化碳中毒（碳氧血红蛋白血症）或高铁血红蛋白血症，SaO₂可能假性正常，此时即使SaO₂≥92%，也应结合临床症状考虑动脉血气分析。
4. **决策输出**：
   - 若满足触发条件（未吸氧 + SaO₂ < 92%），则执行动脉血气分析；
   - 否则维持当前监测方案，并记录观察结果。

> **注意**：SaO₂仅反映血红蛋白氧合比例，不能代表总氧含量或组织氧利用情况。动脉血气分析是诊断缺氧的金标准。
