# 血小板聚集动力学分析

## 适用对象
- 血小板悬液
- 富含血小板血浆（PRP）
- 各类致聚剂（如ADP、胶原、凝血酶）

## 前提条件
确保以下要素存在：
- Ca²⁺
- 纤维蛋白原
- GPⅡb/Ⅲa受体可被活化

## 执行步骤

1. **确认实验体系完整性**：验证样本中Ca²⁺、纤维蛋白原充足，且无GPⅡb/Ⅲa缺陷或低纤维蛋白原血症。

2. **施加致聚剂刺激**：向血小板样本中加入ADP、胶原或凝血酶等致聚剂，启动聚集反应。

3. **监测光密度变化**：通过血小板聚集仪记录透光度随时间变化，识别聚集时相模式：
   - **第一时相**：快速、可逆，由外源性致聚剂直接引发；表现为透光度先升高后下降（解聚）。
   - **第二时相**：缓慢、不可逆，由血小板释放的内源性ADP驱动；透光度持续升高。

4. **根据致聚剂浓度判读曲线类型**：
   - 低浓度ADP → 仅第一时相（曲线A）
   - 中浓度ADP → 第一+第二时相（曲线B）
   - 高浓度ADP → 单一不可逆高峰（曲线C）
   - 胶原 → 经潜伏期后直接引发第二时相（曲线D）

5. **评估调控机制影响**：
   - 若使用阿司匹林预处理，预期TXA₂生成减少，第二时相减弱或消失。
   - 若存在内皮源性PGI₂或NO，应观察到聚集抑制（cAMP/cGMP升高）。

6. **输出结果解释**：结合聚集曲线形态、时相特征及药物干预情况，判断血小板功能状态或药物效应。

> 注意：本技能不适用于GPⅡb/Ⅲa缺陷（如血小板无力症）或低纤维蛋白原血症患者样本。
