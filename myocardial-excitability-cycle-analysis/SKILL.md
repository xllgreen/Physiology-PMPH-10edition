# 心肌细胞兴奋性周期性变化分析

## 适用条件
- 细胞类型：仅适用于心室肌细胞和快反应心肌细胞
- 前提：细胞已完成一次动作电位去极化并处于复极化阶段
- 不适用于慢反应细胞（如窦房结P细胞）

## 执行步骤

1. **确定当前膜电位范围**，对照以下阶段划分：
   - **有效不应期（ERP）**：从0期开始至复极至-60 mV
     - 绝对不应期（ARP）：0期至-55 mV，任何刺激均不能引发新动作电位
     - 局部反应期（LRP）：-55 mV 至 -60 mV，阈上刺激仅引起局部反应
   - **相对不应期（RRP）**：-60 mV 至 -80 mV，阈上刺激可引发动作电位，但兴奋性降低
   - **超常期（SNP）**：-80 mV 至 -90 mV（静息电位），略低于阈值的刺激即可引发动作电位

2. **评估钠通道状态**：
   - 膜电位 > -60 mV：钠通道大部分失活，无法激活
   - -60 mV ~ -80 mV：部分钠通道恢复，但数量不足
   - < -80 mV：钠通道基本恢复至静息（备用）状态

3. **判断可激发性与动作电位特征**：
   - ERP：不可激发
   - RRP/SNP：可激发，但0期去极化速度和幅度减小，传导减慢

4. **解释生理意义**：
   - 长有效不应期防止心肌强直收缩，保障泵血功能
   - RRP和SNP是心律失常（如早搏、折返）的易感窗口

> 提示：若用户提供具体膜电位值或刺激条件，直接返回对应阶段名称、可激发条件及动作电位特征。
