# 突触可塑性机制分析

## 何时使用
- 用户研究学习与记忆的细胞机制
- 需要判断特定刺激频率（如1Hz或50Hz）对突触效能的影响
- 分析海马CA1区、Schaffer侧支、苔藓纤维或感觉运动反射通路中的可塑性类型
- 解释行为适应现象（如习惯化或敏感化）的突触基础

## 执行步骤

### 1. 判断可塑性时间尺度
- **短时程可塑性**：持续数秒至数小时，不依赖蛋白质合成
- **长时程可塑性**：持续数小时以上，常伴随结构重塑与蛋白质合成

### 2. 根据刺激模式识别可塑性类型
- **高频强直刺激（如50Hz）** → 检查是否满足LTP条件（Ca²⁺大幅升高、AMPA受体磷酸化/插膜）
- **低频刺激（如1Hz）** → 检查是否诱导LTD（轻度Ca²⁺升高、AMPA受体去磷酸化/内吞）
- **重复温和刺激** → 考虑习惯化（突触前Ca²⁺通道失活）
- **伤害性刺激后的行为增强** → 考虑敏感化（神经调质介导的突触易化）

### 3. 确认适用通路与机制类型
- **Schaffer侧支（海马CA1）**：LTP主要为突触后机制，依赖NMDA受体和CaMKII
- **苔藓纤维**：LTP可能为突触前机制，涉及cAMP和Ih通道
- **感觉运动反射通路（如海兔喷水管）**：适用于习惯化与敏感化分析

### 4. 验证前提条件
- 确保存在功能性突触
- 确认Ca²⁺信号通路完整
- 刺激模式符合频率约束（LTP需高频，LTD需低频）

### 5. 输出结果
评估突触效能变化的类型（增强/抑制）、持续时间（短时/长时）及潜在行为意义（如记忆编码、适应性反应）。
