# 突触可塑性详细机制参考

## 短时程可塑性

### 强直后增强（PTP）
- **触发条件**：短串高频刺激（如100 Hz，持续1秒）
- **机制**：突触前末梢Ca²⁺蓄积 → 激活蛋白激酶C（PKC）→ 增强囊泡释放概率
- **持续时间**：数分钟至1小时

### 习惯化（Habituation）
- **经典模型**：海兔喷水管-鳃缩反射
- **机制**：反复轻触 → 突触前电压门控Ca²⁺通道失活 → Ca²⁺内流减少 → 递质（如谷氨酸）释放下降
- **恢复**：休息后可逆

### 敏感化（Sensitization）
- **触发**：尾部电击等伤害性刺激
- **机制**：感觉神经元释放5-HT → 激活腺苷酸环化酶 → cAMP↑ → PKA激活 → K⁺通道磷酸化关闭 → 动作电位时程延长 → Ca²⁺内流增加 → 递质释放增多
- **交叉敏感化**：可泛化至其他反射通路

## 长时程可塑性

### LTP（长时程增强）

#### Schaffer侧支通路（CA3→CA1）
1. **诱导**：
   - 高频刺激（如100 Hz × 1s 或 θ爆发刺激）
   - 谷氨酸大量释放 → AMPA受体激活 → 去极化 → Mg²⁺从NMDA受体通道移出
   - NMDA通道开放 → Ca²⁺内流
2. **表达**：
   - Ca²⁺结合钙调蛋白 → 激活CaMKII
   - CaMKII磷酸化AMPA受体GluA1亚基（Ser831）→ 电导增加
   - 促进AMPA受体从胞内池插入突触后膜
   - 可能通过NO或花生四烯酸作为逆行信使增强突触前释放
3. **维持**：
   - 早期LTP（<3小时）：不依赖蛋白质合成
   - 晚期LTP（>3小时）：需CREB激活与新蛋白合成，伴随树突棘增大

#### 苔藓纤维通路（DG→CA3）
- **NMDA非依赖型LTP**
- 机制涉及突触前cAMP/PKA通路和超极化激活阳离子通道（Ih）
- 表现为突触前释放概率增加

### LTD（长时程抑制）
- **诱导**：低频刺激（1Hz × 15分钟）
- **机制**：
  - 中等Ca²⁺升高 → 激活蛋白磷酸酶（如PP1、calcineurin）
  - AMPA受体去磷酸化（GluA1 Ser845位点）→ 电导下降
  - 触发网格蛋白介导的内吞 → AMPA受体数量减少
- **功能**：突触“修剪”，参与记忆精细调节

## 关键变量说明

| 变量 | 类型 | 作用 |
|------|------|------|
| 刺激频率 | 枚举 | 1Hz → LTD；50–100Hz → LTP |
| Ca²⁺浓度变化幅度 | 分类 | 大幅↑ → 激酶激活（LTP）；轻度↑ → 磷酸酶激活（LTD） |
| AMPA受体状态 | 状态变量 | 磷酸化/去磷酸化、膜表面密度决定突触强度 |

## 约束与例外
- 并非所有突触都表现LTP/LTD（如某些抑制性突触）
- 发育阶段影响可塑性阈值（幼年更易诱导LTP）
- 神经调质（如多巴胺、乙酰胆碱）可调节可塑性方向
