# 有效滤过压计算

## 何时使用
- 需要判断液体在毛细血管与组织间隙之间是净滤出还是净重吸收
- 分析心力衰竭、低蛋白血症、炎症等导致水肿的机制
- 评估特定组织（如肌肉、肝脏）的液体交换能力
- **注意**：仅适用于连续型毛细血管，不适用于肾小球等特殊滤过结构（除非特别说明）

## 执行步骤
1. **确认四种压力值**：
   - 毛细血管血压（动脉端约30–40 mmHg，静脉端约10–15 mmHg）
   - 组织液胶体渗透压（通常约5 mmHg）
   - 组织液静水压（通常为−3 至 0 mmHg）
   - 血浆胶体渗透压（正常约25 mmHg，主要由白蛋白决定）

2. **代入有效滤过压公式**：
   ```
   EFP = (毛细血管血压 + 组织液胶体渗透压) − (组织液静水压 + 血浆胶体渗透压)
   ```

3. **解释结果**：
   - 若 EFP > 0：发生净滤过，液体从毛细血管进入组织间隙
   - 若 EFP < 0：发生净重吸收，液体从组织间隙返回毛细血管

4. **（可选）估算液体流量**：
   - 使用公式：单位时间滤过量 = EFP × 滤过系数（K_f）
   - K_f 因组织而异（脑和肌肉较小，肝和肾较大）

5. **关联临床意义**：
   - 结合EFP结果判断是否存在水肿风险
   - 解释为何某些疾病（如低蛋白血症）导致组织液积聚
