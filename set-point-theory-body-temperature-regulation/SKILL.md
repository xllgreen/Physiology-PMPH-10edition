# 调定点学说与体温调节

## 何时使用
- 分析发热患者出现畏寒、寒战或出汗的原因
- 判断体温升高是否由调定点改变引起（如感染、炎症）
- 区分调节性发热与非调节性体温升高（如中暑、中枢损伤）

## 执行步骤

1. **确认适用对象**：仅适用于恒温动物、发热患者或处于感染/炎症状态的个体；不适用于中暑等非调节性体温升高。

2. **评估调定点是否发生重调定**：
   - **调定点上移**（常见于致热原作用）：
     - 热敏神经元反应曲线斜率减小（W→W'），或
     - 冷敏神经元反应曲线斜率增大（C→C'）
   - **调定点下移**：
     - 热敏神经元斜率增大，或
     - 冷敏神经元斜率减小

3. **比较实际体温与新调定点**：
   - 若实际体温 < 新调定点 → 启动产热↑、散热↓机制 → 表现为畏寒、寒战
   - 若实际体温 > 新调定点 → 启动产热↓、散热↑机制 → 表现为出汗、皮肤血管扩张

4. **排除非调节性体温升高**：若无调定点改变（如中暑因散热障碍导致），则不属于本机制范畴。

> 注意：调定点重调定依赖PO/AH区热敏与冷敏神经元放电特性的改变，且需存在致热原或其他调节因子。
