# 载体介导的易化扩散

## 何时使用
当需分析葡萄糖、氨基酸或果糖等水溶性小分子如何顺浓度梯度进入细胞，或评估组织摄取能力（如胰岛素抵抗）时使用本技能。

**触发条件**：物质为水溶性小分子，存在浓度梯度，且细胞表达相应特异性载体蛋白（如GLUT1–5）。

## 执行步骤
1. **确认底物与载体匹配**：验证物质是否为载体的特异性底物（如GLUT仅转运右旋葡萄糖）。
2. **检查浓度梯度方向**：确保转运方向为从高浓度向低浓度。
3. **执行四步转运循环**：
   - 载体在高浓度侧结合底物
   - 发生构象变化，封闭底物
   - 构象转向低浓度侧
   - 底物解离释放
4. **评估关键特性**：
   - **饱和现象**：当底物浓度达K_m时速率达V_max的一半；继续增加浓度不再提升速率
   - **竞争性抑制**：结构相似物（如果糖与葡萄糖）可相互抑制
5. **关联生理场景**：
   - GLUT4在肌肉/脂肪组织中受胰岛素调控
   - GLUT2在肝细胞中实现双向转运
   - GLUT5专司果糖转运

## 输出结果
返回底物跨膜转运速率，用于评估组织代谢能力或诊断转运障碍（如糖尿病中的胰岛素抵抗）。
