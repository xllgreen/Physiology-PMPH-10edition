# 化学感受性呼吸反射的双重通路机制

## 触发条件
当动脉血或脑脊液中出现以下任一变化时激活本机制：
- PO₂ < 80 mmHg
- PCO₂ 升高
- H⁺ 浓度升高（pH 下降）

## 执行步骤

### 1. 判断适用前提
- 确认血-脑屏障完整
- 确认外周化学感受器传入神经未被切断（如未行颈动脉体切除术）
- 排除严重脑损伤导致中枢化学感受器失效的情况

### 2. 识别主导通路
- **若主要刺激为低氧（PO₂ < 80 mmHg）**：仅外周化学感受器通路有效（因中枢直接受低氧抑制）
- **若主要刺激为高PCO₂或H⁺升高**：优先考虑中枢化学感受器通路（贡献约80%通气反应）
- **若多种刺激共存**：评估协同与拮抗效应（见步骤3）

### 3. 分析通路交互
- CO₂升高 → 脑脊液H⁺升高 → 强烈兴奋中枢通路
- H⁺升高 → 通气增强 → CO₂排出 → PCO₂下降 → 部分抵消原刺激
- 低O₂ → 通气增强 → CO₂排出 → PCO₂和H⁺下降 → 削弱低氧驱动

### 4. 评估临床风险
- 对慢性高碳酸血症患者（如肺气肿），呼吸主要依赖低氧对外周感受器的刺激
- 此类患者吸入高浓度氧可能解除呼吸驱动，导致急性呼吸抑制

### 5. 输出判断
- 明确激活的通路（外周、中枢或两者）
- 预测通气反应强度（参考变量阈值）
- 标注潜在临床警示

> 详细机制、阈值数据及适应现象说明见 `references/dual-pathway-mechanism-details.md`。
