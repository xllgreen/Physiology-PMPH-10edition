# 血红蛋白氧结合特性分析

## 何时使用
- 分析肺部（高PO₂）或组织（低PO₂）中Hb与O₂的结合或释放
- 计算血氧相关指标（氧容量、氧含量、氧饱和度）
- 解释为何严重贫血或CO中毒时缺氧但不发绀

## 如何执行
1. 确认适用条件：Hb为正常二价铁状态（Fe²⁺），无CO中毒或高铁血红蛋白。
2. 获取关键参数：
   - Hb浓度（g/100ml）
   - 当地PO₂（mmHg）
3. 计算指标：
   - **氧容量** = Hb浓度 × 1.34 ml O₂/g Hb
   - **氧含量** = Hb浓度 × 1.34 × (氧饱和度 / 100)
   - **氧饱和度** 可通过标准氧离曲线估算（动脉血PO₂=100 mmHg时≈97.5%；静脉血PO₂=40 mmHg时≈75%）
4. 判断临床意义：
   - 去氧Hb ≥5 g/100ml 时出现发绀
   - 血浆溶解O₂可忽略，故上述指标代表总血氧状态

> 注意：该过程为“氧合”而非“氧化”，Fe²⁺价态不变。
