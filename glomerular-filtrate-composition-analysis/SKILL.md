# 肾小球滤液成分特征分析

## 何时使用
- 需要说明肾小球滤液为何被称为“血浆的超滤液”
- 比较超滤液与血浆中溶质（如葡萄糖、尿素、电解质）的浓度差异
- 判断滤过膜是否正常工作（例如蛋白质是否异常漏出）
- 排除病理状态干扰前的生理学基准分析

## 执行步骤
1. **确认前提条件**：血液正流经肾小球毛细血管，且滤过膜结构与功能正常。
2. **识别可滤过成分**：血浆中除蛋白质外的所有小分子溶质（如葡萄糖、氯化物、无机磷酸盐、尿素、尿酸、肌酐）均可自由通过滤过膜进入肾小囊腔。
3. **验证理化性质**：超滤液的渗透压、pH值及上述溶质浓度与血浆高度一致。
4. **排除蛋白质**：因分子量大且带负电荷，血浆蛋白被滤过膜有效阻挡，超滤液中蛋白质浓度极低（通常可忽略）。
5. **得出结论**：在正常生理条件下，肾小球滤液 = 血浆 − 蛋白质，此关系可用于评估滤过屏障功能。

> ⚠️ 注意：本技能仅适用于滤过膜通透性正常、血浆蛋白浓度无显著异常的生理状态。若存在肾病综合征、糖尿病肾病等病理情况，需结合其他诊断技能。
