# 电压门控钠钾通道动态特性与动作电位形成

## 何时使用
- 需要解释动作电位快速上升支和复极化下降支的成因
- 分析河豚毒素（TTX）或四乙铵（TEA）等药物对电活动的影响
- 理解阈电位触发机制及正反馈循环
- 区分Na⁺与K⁺通道的功能状态差异

## 核心操作流程
1. **确认前提条件**：
   - 细胞表达电压门控Na⁺/K⁺通道
   - 处于静息状态（约-70 mV）
   - 接受去极化刺激
2. **识别阈电位触发点**：当膜电位去极化至约-55 mV时，启动动作电位。
3. **追踪Na⁺通道动态**：
   - 激活门（m门）快速打开 → Na⁺电导（G_Na）剧增 → Na⁺内流 → 进一步去极化（正反馈）
   - <1毫秒后失活门（h门）关闭 → 通道失活 → Na⁺内流终止
4. **追踪K⁺通道动态**：
   - 激活门（n门）延迟打开（约1毫秒）→ K⁺电导（G_K）缓慢上升
   - K⁺外流加速复极化
5. **预测阻断剂效应**：
   - TTX阻断Na⁺通道 → 动作电位幅度下降或消失
   - TEA阻断K⁺通道 → 复极化延迟，动作电位时程延长

> 注意：本机制不适用于仅依赖Ca²⁺通道的动作电位（如某些平滑肌）。
