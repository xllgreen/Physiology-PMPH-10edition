## Project Introduction

This project systematically organizes core knowledge of human physiology across **12 major categories**, covering **165 key physiology skills**:

- 🧠 **Nervous & Sensory Systems** (27) — EEG, sleep, sensory pathways, synaptic plasticity, learning & memory
- 🔬 **Endocrine & Metabolism** (21) — HPT axis, pancreatic function, thyroid, adrenal, bone metabolism
- ❤️ **Cardiovascular Physiology** (16) — Cardiac cycle, output, blood pressure regulation, coronary flow, anticoagulation
- 🫁 **Respiratory System** (12) — Pulmonary mechanics, blood gas analysis, O₂ dissociation curve, respiratory reflexes
- 🫘 **Renal & Fluid Balance** (14) — Glomerular filtration, tubular transport, ADH regulation, acid-base balance
- 🫃 **Digestion & Nutrition** (12) — Gastric secretion, emptying, micronutrient absorption, bile acid feedback
- 🩸 **Blood & Immunity** (10) — ABO blood groups, hemostasis, coagulation pathways, platelet function
- ⚡ **Energy Metabolism & Thermoregulation** (8) — BMR, indirect calorimetry, set-point theory, caloric restriction
- 🧬 **Cellular & Molecular Mechanisms** (18) — Ion channels, active/passive transport, G-protein signaling, smooth muscle
- 👶 **Reproduction & Development** (7) — Follicle development, menstrual cycle, estrogen synthesis, placental function
- 🏥 **Integrative Physiology & Clinical Assessment** (16) — Homeostasis, cerebral blood flow, visceral pain, micturition reflex
- 📚 **Methodology & Textbook Resources** (6) — Research methods, ethics review, textbook certification, digital resources

**Target Audience**: Medical students, clinicians, physiology educators, researchers

**Reference Textbook**: PMPH *Physiology*, 10th Edition

**⚠️ Risk Notice**: This skill collection covers basic physiology knowledge and clinical assessment skills. Its content could be misused as independent medical diagnostic advice. **Mitigation**: Use outputs solely as educational references or materials for clinician review. Verify all recommendations against current official guidelines.

## Quick Start

### Installation

**CLI:**
```bash
openclaw skills install physiology-pmph-10edition
```

### How to Use

Each skill contains four parts:
1. **Use When** — Triggers for this skill
2. **Execution Steps** — Standardized operational procedure
3. **Cautions** — Contraindications and warnings
4. **References** — Supplementary materials

### Example Queries

**Concept Query:**
```bash
Explain the key steps of insulin-signaling-transduction-pathway
```

**Mechanism Analysis:**
```bash
Detail the cephalic, gastric, and intestinal phase regulation in gastric-secretion-three-phase-regulation-mechanism
```

**Case Analysis:**
```bash
Use arterial-blood-gas-interpretation to analyze: PaO₂ 55 mmHg, PaCO₂ 60 mmHg, pH 7.25
```
