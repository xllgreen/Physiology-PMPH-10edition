# ERP/APD比值分析用于抗心律失常作用评估

## 何时使用
- 分析某药物或生理/病理状态对抗心律失常的作用机制；
- 需要判断该干预是否通过改变心肌细胞的不应期特性来抑制折返；
- 研究对象为心室肌细胞、心房肌细胞等快反应非自律性心肌细胞；
- 已获得可测量的ERP和APD数据。

## 如何执行
1. **确认适用条件**：
   - 确保研究对象为心室肌或心房肌细胞（不适用于窦房结等慢反应自律细胞）；
   - 确认心律失常机制为折返性。

2. **获取关键参数**：
   - 测量有效不应期（ERP，单位：ms）；
   - 测量动作电位时程（APD，通常指APD90，单位：ms）。

3. **计算ERP/APD比值**：
   - 使用公式：`ERP_APD_ratio = ERP / APD`。

4. **解释结果**：
   - 若比值增大（无论ERP和APD绝对值如何变化），提示具有抗心律失常潜力；
   - 若比值减小，则可能促发心律失常。

5. **输出结论**：
   - 明确说明ERP/APD比值的变化方向（增大/减小）；
   - 判断是否具有抗心律失常效应。

> 注意：即使ERP和APD同步延长或缩短，也必须分别评估其相对变化程度。
