# 药物实例与机制详解

## 奎尼丁（Ⅰ类抗心律失常药）
- 同时延长ERP和APD；
- ERP延长程度 > APD延长程度；
- 结果：ERP/APD比值增大 → 抑制折返 → 抗心律失常。

## 利多卡因
- 同时缩短ERP和APD；
- ERP缩短程度 < APD缩短程度；
- 结果：ERP/APD比值仍增大 → 抗心律失常。

## 临床意义补充
- 在长QT综合征等不应期离散度高的情况下，提高ERP/APD比值有助于稳定心肌电活动；
- ERP反映钠通道恢复能力，APD主要反映钾通道介导的复极化速度；
- ERP定义：从动作电位0期开始至膜电位恢复至-60 mV的时间点；
- APD通常采用APD90（复极至90%的时间）作为标准指标。
