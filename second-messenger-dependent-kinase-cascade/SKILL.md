# 第二信使依赖性蛋白激酶的磷酸化级联机制

## 适用场景
- 解释第二信使如何激活PKA、PKC、CaMK、PKG等蛋白激酶
- 分析底物蛋白磷酸化后的功能变化（如酶活性、离子通道开闭、转录因子核转位）
- 阐述信号级联放大与去磷酸化终止机制

## 执行步骤
1. **第二信使生成**：GPCR或酶联受体激活效应器酶（如AC、PLC），产生cAMP、IP₃、DG、Ca²⁺等第二信使。
2. **蛋白激酶激活**：
   - cAMP → 激活PKA
   - Ca²⁺ + 钙调蛋白 → 激活CaMK或钙调磷酸酶
   - DG + Ca²⁺ + 磷脂酰丝氨酸 → 激活PKC
   - cGMP → 激活PKG
3. **磷酸化反应**：激酶将ATP的磷酸基团转移至底物蛋白的丝氨酸/苏氨酸（PKA/PKC/PKG）或酪氨酸（RTK）残基。
4. **功能改变**：磷酸化导致蛋白构象、电荷、活性或定位变化，影响代谢、基因表达或膜电位。
5. **级联放大**：若底物为激酶，则触发多级磷酸化级联，显著放大信号。
6. **信号终止**：蛋白磷酸酶（PP1、PP2A、钙调磷酸酶）催化去磷酸化，恢复底物原始状态。

## 注意事项
- 底物必须含有可磷酸化氨基酸残基
- 不适用于非磷酸化调控的信号通路
- cAMP还可通过EPAC或HCN通道介导非经典通路
