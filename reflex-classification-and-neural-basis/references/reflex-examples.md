# 反射分类示例

## 非条件反射
- **屈肌反射**：肢体遇伤害性刺激回缩，脊髓介导
- **瞬目反射**：异物接近眼睛时眨眼，脑干介导
- **性反射**：生殖器刺激引发勃起，脊髓介导

## 条件反射
- **经典条件反射**（巴甫洛夫）：铃声（中性刺激）与食物（非条件刺激）多次配对后，单独铃声引发唾液分泌
- **操作条件反射**：动物按压杠杆获得食物，形成主动行为

## 关键区别
| 特征 | 非条件反射 | 条件反射 |
|------|------------|----------|
| 先天性 | 是 | 否 |
| 中枢 | 皮层下 | 大脑皮层 |
| 可塑性 | 固定 | 可建立/消退 |
| 数量 | 有限 | 无限 |
